## Mechanism Statement

耐心资本通过提供市场化长期资金，降低企业对政府短期刺激与补贴的依赖，缓解地方政府干预与财政补贴对资源配置的扭曲（policy distortion，ID），进而提升企业产能利用率。这一路径被何文彬 (2025) 称为耐心资本的"免疫效应"。

## Theoretical Logic

中国式产能过剩的一条经典解释是体制扭曲：地方政府为追求政绩、就业或税收，倾向通过财政补贴、信贷扶持、土地优惠等手段引导资本进入特定行业；这种非市场信号驱动的资源配置造成行业内同质化重复投资，引发"潮涌现象"与产能过剩。耐心资本作为不依赖政府短期政策周期的长期资金来源，能在两个层面缓解这一扭曲：

- 替代效应：耐心资本提供稳定融资，降低企业对政策红利的依赖；企业重新关注市场需求与长期战略。
- 资源竞争效应：耐心资本以市场化绩效评估为筛选机制，反向迫使政策性资金分配向市场逻辑收敛，减少低效产能存续。

二者共同构成"政策扭曲修复"路径，使资本配置从"政策驱动型"过渡到"市场驱动型"。

## Empirical Proxy

何文彬 (2025) 用 ID = (一般预算支出 / GDP) × (政府补助 / 总资产) 衡量市场机制的政策扭曲，捕捉宏观政府干预度 × 微观补贴强度的交叉。再做分位数归一化得到 [0, 1] 区间变量。Sobel 中介检验路径：

1. PC → ID（PC_Sd 与 PC_Rl 系数应显著为负，PC_Rl 抑制效应更强）。
2. ID → CU（ID 系数应显著为负）。
3. PC + ID 同回归，PC 仍显著但系数衰减，ID 显著为负 → 部分中介。

## Evidence Across Papers

仅 [[何文彬-2025-耐心资本-产能利用率]] 一篇报告：表 6 中 PC_Sd → ID 系数 -0.0191（5%），PC_Rl → ID 系数 -0.1501（1%）；ID → CU 系数 -0.0099、-0.0087（均 1%）；Sobel 统计量 2.15（PC_Sd）与 8.53（PC_Rl）。两类耐心资本均通过中介检验，PC_Rl 路径更显著。

## Boundary Conditions

- 在过剩产能行业中可能失效：何文彬 (2025) 异质性分析显示，过剩产能行业子样本（C22、C25、C26、C28、C30、C31、C32 等）耐心资本对 CU 反向作用，可能因 ID 改善后反而加剧"固化效应"。
- 宏观政府干预度的衡量方式（一般预算支出 / GDP）属粗粒度代理；若改用具体政策密度指标，结果可能变化。
- 国有企业在政策扭曲下表现复杂；耐心资本对 SOE 子样本的 CU 提升作用本身不显著。

## Open Questions

- 政策扭曲指标 ID 的分位数归一化是否会扭曲中介效应估计？是否需要做不归一化对照？
- PC_Sd 与 PC_Rl 在缓解政策扭曲上效应大小相差近 8 倍，债权侧的"长期负债占比"是否含其它机制成分（如银行信贷的信号效应）？
- 该机制能否用 IV 进一步识别？目前论文仅用逐步回归 Sobel，未做反事实识别。
