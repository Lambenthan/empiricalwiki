# 耐心资本测算方法清单

## 一句话回答

EmpiricalWiki 当前涵盖 **8 种主操作化框架（A1–A4、B1、B2、C、D）**，叠加债权侧分子分母口径差异与"是否求和"两个维度，可枚举出**至少 11 条不同的具体测算路径**。下文按"框架 + 子变体 + 来源论文"逐一列出。

## 详细变体卡片

### A1 经典版（换手率 3 分组 + 总长债）

**测算口径**：

```
Step 1：机构投资者每半年累计买入与卖出股票市值中较小者
        ÷ 半年内持有股票平均市值 = 机构 k 的半年度换手率 CR_kt
Step 2：研究期内 4 个或多个半年度求平均 → AVR_CR_k
Step 3：按 AVR_CR_k 升序等距三分位，最低组定义为稳定型机构投资者
Step 4：稳定型股权 (Equity / Stequity / Wd) = 稳定型机构在公司 i 的合计持股比例
Step 5：关系型债权 (Debt / Rdebt / Pc) = 长期负债 / 负债总额
        其中长期负债 = 长期借款 + 应付债券 + 长期应付款（不同论文细分项略有差异）
Step 6：PC = Equity + Debt（求和），或两者分别建模并报告分项系数
```

**数据源**：[[csmar]] 机构投资者持股表（换手率所需买卖明细需补 [[wind]] / 同花顺）+ CSMAR 资产负债表。

**优点**：
- 中文耐心资本研究中曝光率最高、最被审稿人认可的"标配"测度。
- 同时覆盖股权侧时间属性 + 债权侧关系属性，构念覆盖最完整。

**风险**：
- 换手率三分组丢弃中间组样本（约 1/3 机构）；公司层 Stequity + 交易型不等于全口径机构持股。
- 周绍妮 (2017) 表 2 显示中国稳定型机构投资者中指数型基金占 77.49%，"稳定 ≈ 被动持有"，与西方文献"稳定 = 主动监督"假设方向相反，需在论文中明示。
- 换手率原始字段需要机构-股票-期末持股 + 期末股价 + 上一期持股变化的完整面板，[[csmar]] 缺失部分，需用 [[wind]] / 同花顺补齐。

**项目可用性**：股权侧需获取机构投资者每半年持股明细 + 价格，[[wind]] 接入门槛较高；债权侧 [[csmar]] 直接可得。**部分可得**。

**相关变量页**：[[stable-institutional-investors-turnover]] · [[transactional-institutional-investors-turnover]] · [[relational-debt-total-long-debt-ratio]]。

---

### A1-仅股权（简化版）

**测算口径**：与 A1 完全相同，但主回归只放股权侧（长期机构投资者持股比例），债权侧（如 Bank = 银行长贷 / 应付债券 + 应付票据 + 银行长贷）放在稳健性章节作"替换核心解释变量"。

**典型论文**：李思飞 2025、温磊 2024、贾勇 2025。

**优点**：避免债权侧口径争议，主回归更干净。
**风险**：放弃了构念中"债权关系属性"维度，存在测度遗漏。
**项目可用性**：需求最低，仅需股权侧数据。

---

### A2 标准差变体 + 银行长债

**测算口径**：

```
稳定型股权 = 当期机构投资者整体持股比例 ÷ 过去 3 年持股比例标准差
关系型债务占比 = 银行长期贷款 ÷ (银行长期贷款 + 应付债券 + 应付票据)
PC = 稳定型股权 + 关系型债务（求和合成）
```

**数据源**：[[csmar]] 机构投资者持股 + 长期借款明细（需手工拆出银行长贷部分，建议接 [[wind]] 补齐分项）。

**优点**：
- 保留连续值，便于 IV / 非线性 / 交互项检验。
- 银行长贷分子比"总长期负债"更贴近"关系型"构念（关系型 ≈ 长期合作的银企关系）。

**风险**：
- 持股比例 / 三年标准差，分母 → 0 时极不稳定，小公司或上市不足 3 年样本极易爆炸；需缩尾或加 ε。
- 银行长贷在 [[csmar]] 中需从长期借款明细手工识别，自动化难度较高。
- 股权和债权两个不同量纲直接相加，加和的经济意义偏弱。

**项目可用性**：稳定型股权可由 [[csmar]] 计算；银行长贷需补 [[wind]]。**部分可得**。

**相关变量页**：[[patient-capital]]（主页面以 A2 为基准）。

---

### A3 标准差变体 + 非流动负债明细

**测算口径**：

```
SD = 当期机构持股比例 / 过去 3 年持股比例标准差
DE = (长期借款 + 应付债券 + 长期应付款 + 其他非流动负债) / 负债总额
两变量并列进入回归，不合成
```

**典型论文**：[[简冠群-2025-专精特新-新质生产力-风险投资-耐心资本]]。

**优点**：
- 与 A2 共享股权侧 SD 连续指标，方法学可比。
- 债权侧不挑剔银行借款，分子覆盖更全面，[[csmar]] 资产负债表可直接拼出。

**风险**：
- 与 A2 的债权侧不可直接换算；同一公司在 A2 / A3 下"耐心资本"判定可能差异显著。
- 包含非银金融机构借款（信托、险资等），与"关系型"构念的银企关系含义偏离。

**项目可用性**：[[csmar]] 直接可得，**完全可得**。

---

### A4 换手率 2 分组 + 总长债

**测算口径**：

```
机构投资者按每半年度平均换手率与"全样本换手率均值"比较：
   高于均值 = 交易型；低于均值 = 稳定型（无中间组）
Invest = 稳定型机构在公司 i 的合计持股比例
Debt = 长期负债 / 总负债
两变量并列建模，不合成
```

**典型论文**：[[强国令-2025-耐心资本-漂绿]]。

**优点**：
- 二分位避免丢中间样本，所有机构都参与公司层汇总。
- 阈值是"全样本均值"而非"行业-年度中位数"，跨行业可比性更强。

**风险**：
- 与 A1 三分组无法直接换算；同一机构在 A1 中是"中间组（弃用）"，在 A4 中可能落入稳定或交易型。
- 单一阈值不分行业，可能掩盖行业内换手率分布异质性。

**项目可用性**：[[csmar]] / [[wind]] 机构持股明细 + 资产负债表，**部分可得**（仍需机构买卖额数据）。

---

### B1 持股时长法

**测算口径**：

```
对每个机构投资者 s 在每家公司 i 计算：
   T_si = 累积持有天数（或累积持有期数）
对每个机构 s 计算其投资过的所有公司平均时长：
   T_s = (1 / n_s) Σ T_si
若 T_si ≥ T_s，标记机构 s 为公司 i 的战略型 / 长期机构投资者
PC_Sd_i = 公司 i 的战略型机构投资者持股比例之和

[何文彬 2025 同时定义]
PC_Rl = 长期负债 / 负债总额（简化版关系型债权）
两变量并列进入回归
```

**典型论文**：[[何文彬-2025-耐心资本-产能利用率]]、[[谢婷婷-2025-耐心资本-动态能力-绿色转型]]。

**优点**：
- 用"持有时长"直接刻画"耐心"，比换手率间接指标更接近构念字面含义。
- 不需要价格数据，仅依赖持股明细的起止日期。

**风险**：
- 机构投资者代码合并、并购重组导致持仓不连续，时长口径敏感。
- 阈值是"该机构自身平均时长"，机构间不可比；"持仓 4 年（已是长期）"与"持仓 1 年（为该机构最长）"都可能被判为战略型。
- 与 A1/A2/A4 的"换手率 / 标准差"路径在数值上不可换算。

**项目可用性**：[[csmar]] 机构投资者持股表带季度起止日期，**完全可得**。

---

### B2 持股稳定性中位数法

**测算口径**：

```
Step 1：SD_{i,t} = INVH_{i,t} / STD(INVH_{i,t-3..t-1})
        分子是当期机构持股比例，分母是过去 3 年（不含本期或含本期，不同论文有差异）的标准差
Step 2：按行业-年度计算 SD 中位数 MEDIAN(SD)_{ind, year}
Step 3：INVW = 1 if SD_{i,t} ≥ MEDIAN；否则 INVW = 0
        INVW = 1 标记稳定型机构投资者样本组

[李季鹏 2025 同时定义]
Equity = 行业-年度中位数门槛下的战略型机构持股比例之和（连续值，非 0/1）
Debt = 长期负债 / 负债总额
两变量并列建模
```

**典型论文**：[[徐灿宇-2023-异质机构投资者-实地调研-财务重述]]、[[杨棉之-2020-机构投资者异质性-卖空机制-股价崩盘风险]]、[[李季鹏-2025-耐心资本-高质量发展]]。

**优点**：
- 行业-年度中位数门槛控制了行业内换手率分布异质性。
- 输出二值变量（INVW）便于 Logit / PSM；输出连续 Equity 也兼容 OLS。
- 阈值"过中位数"让稳定型与交易型大致各占一半，样本均衡。

**风险**：
- 二值化丢失连续信息：A2 框架可保留连续 SD，B2 转哑变量后判别力受影响。
- 行业分类口径影响中位数（CSRC 大类 vs 制造业二级 vs 申万），同一公司不同口径下可能 Stable 反转。
- 与 A2 的连续 SD 在数值上不可直接换算，是否经过中位数二值化是核心区别。
- 公式分母为标准差，命名"stability"但方向是越大越稳定，论文中需明示避免误读。

**项目可用性**：[[csmar]] 机构投资者持股表 + 行业代码，**完全可得**。

**相关变量页**：[[institutional-investor-heterogeneity-stability]] · [[heterogeneous-institutional-investors-stable]]。

### C 综合指标 / 熵值法

**测算口径（邱蓉 2024）**：

```
3 个一级维度 → 7 个二级指标：
  长期导向（Long-term orientation）
  风险承受力（Risk tolerance）
  战略性关系性（Strategic relational）
对所有二级指标做方向标准化（保留正向 / 反向方向）
按客观熵权法计算：信息熵 → 差异系数 → 权重
Pat = 加权合成的综合得分（量纲为 0–1，量级远小于 A 框架的"比值之和"）
```

**典型论文**：[[邱蓉-2024-耐心资本-全要素生产率]]。

**优点**：
- 在单一指标下整合了股权 + 债权 + 时间 + 风险 + 战略多维信息，构念覆盖最广。
- 熵权法是客观赋权，避免主观加权争议。

**风险**：
- 7 个二级指标的具体定义在论文中并未完全公开，复现困难。
- 不同论文的指标体系若不同，"耐心资本得分"在跨研究比较时无法对齐。
- 熵权法对各指标分布敏感，加入或剔除某一指标会改变全部权重。

**项目可用性**：取决于二级指标具体口径，需结合 [[csmar]] + [[wind]] + [[cnrds]] 多源整合，**需要手工整理**。

---

### D 基金持仓视角

**测算口径**：

```
Proportion_{i,j,t} = 基金 i 在 t 年度对企业 j 的持仓 ÷ 基金 i 净值 × 100
（量纲为百分号 %）
长期持有哑变量 LongHold（可选）：基金 i 对企业 j 持有期 ≥ 阈值（如 3 年）
```

**典型论文**：[[李心武-2025-耐心资本-数字化转型-价值发现]]。

**优点**：
- 把"耐心资本"操作化为基金行为本身，避免对"机构是否稳定型"做分类。
- 直接对应"耐心资本 → 价值发现 / 数字化"等以基金为主体的研究问题。

**风险**：
- 单位是 (基金 × 公司 × 年) 的三维面板，而非 (公司 × 年) 二维，与 A/B/C 不可在同一回归框架对齐。
- 仅覆盖公募基金，未覆盖保险、社保、QFII、险资等其他耐心资本主体。
- 与"耐心资本 → 微观企业行为"的主流研究范式衔接较弱。

**项目可用性**：需要基金重仓持股数据（[[csmar]] 基金持仓表 + [[wind]]），**部分可得**。

## 综合判断

如果用最严格的"具体测算路径数"计算，把"是否求和合成"、"债权侧分子分母细分"也作为独立维度，本 wiki 至少识别出以下 11 条不同路径：

1. A1 求和：Stequity + 长债占比 → 单一 PC（唐亮 2025）
2. A1 分项：Stequity 与 Rdebt 分别建模（杨芳 2024 / 杨芳 2025 / 储佩佩 2025）
3. A1 仅股权：仅 Stequity 主回归，债权作稳健性（李思飞 / 温磊 / 贾勇）
4. A2 求和：稳定型股权（标准差比） + 银行长贷占比 → 单一 PC（代飞 2025）
5. A3 分项：SD（标准差比） + DE（非流动负债 / 负债总额）分别建模（简冠群 2025）
6. A4 分项：Invest（换手率 2 分组持股） + Debt（长债占比）分别建模（强国令 2025）
7. B1 求和或分项：PC_Sd（持股时长 ≥ 机构均值）+ PC_Rl（长债 / 总负债）（何文彬 / 谢婷婷）
8. B2 哑变量：INVW = 1 / 0（徐灿宇 2023 / 杨棉之 2020）
9. B2 连续 + 债权：Equity（行业中位数门槛战略型机构持股） + Debt（李季鹏 2025）
10. C 熵值法：3×7 指标客观熵权法合成 Pat（邱蓉 2024）
11. D 基金持仓：Proportion（基金对企业持仓 ÷ 基金净值）（李心武 2025）

如果忽略"是否求和"和"债权侧细节"等次级差异，则**收敛为 8 种主框架（A1、A2、A3、A4、B1、B2、C、D）**，与 [[patient-capital]] 主页面 Measurement 段落完全一致。
