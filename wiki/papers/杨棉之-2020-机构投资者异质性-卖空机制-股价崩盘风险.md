## 研究问题

机构投资者整体持股是抑制还是加剧 A 股主板上市公司股价崩盘风险？将机构投资者按持股稳定性划分为稳定型 vs 交易型后，两类机构投资者对股价崩盘风险的方向性影响是否相反？2010 年融资融券试点提供的卖空机制松绑，是否改变上述两类机构投资者与股价崩盘风险的关系？

## 理论机制

- 隐藏坏消息假说（Jin and Myers, 2006; Hutton et al., 2009）：管理层因机会主义动机刻意囤积负面信息，累积超过临界点后集中释放，导致股价崩盘。
- 机构投资者监督 vs 合谋的双向作用：稳定型机构投资者持股期长、持股比例高，通过参与治理压缩管理层隐瞒坏消息空间；交易型机构投资者持股期短、追求价差，倾向于与管理层合谋掩盖信息或通过羊群行为推升泡沫。
- 卖空机制对悲观信息进入股价的渠道作用（Miller, 1977; Hong and Stein, 2003）：限制卖空时悲观预期无法及时进入股价，泡沫积累放大崩盘风险；松绑后悲观信息可融入股价，且更利于稳定型机构投资者参与治理，加剧两类机构投资者影响崩盘风险的差异。

## 研究假设

- H1：其他条件不变时，机构投资者整体持股加剧上市公司未来股价崩盘风险。
- H2：其他条件不变时，稳定型机构投资者与股价崩盘风险负相关，交易型机构投资者与股价崩盘风险正相关。
- H3：卖空机制减弱交易型机构投资者与股价崩盘风险之间的正向关系，增强稳定型机构投资者与股价崩盘风险之间的负向关系。

## 数据与样本

- 区间：2009-2016 年（INVW 子样本因需 t-3 年滚动标准差，子样本量 4164）。
- 范围：沪深两市 A 股主板上市公司。
- 数据来源：[[csmar]]。机构投资者持股口径包括基金、券商理财、QFII、保险、社保、企业年金、信托、财务公司及其他。
- 筛选规则：剔除金融行业、ST/*ST/PT、年交易周数 < 30、关键变量缺失、当年 IPO 的公司。
- 缩尾：连续变量上下 1% Winsorize。
- 最终样本：8025 个公司-年观测。

## 变量设定

- 被解释变量：[[stock-price-crash-risk]]
  - Ncskew：负收益偏态系数。先按 Dimson 修正回归 r_{i,t} = α + β1 r_{m,t-2} + β2 r_{m,t-1} + β3 r_{m,t} + β4 r_{m,t+1} + β5 r_{m,t+2} + ε_{i,t} 得到残差，再用 W_{i,t}=ln(1+ε_{i,t})。
  - Duvol：收益上下波动率，下行组方差与上行组方差比值取对数。
  - 稳健性测度 IsCrash：W_{i,t} ≤ Average(W_{i,t}) − 3.09 σ_{i,t} 触发即记为 1。
- 核心解释变量：
  - [[institutional-investor-holding]] INVH = 年末机构投资者持股股数 / 总股数。
  - [[institutional-investor-heterogeneity-stability]] INVW：先算 SD_{i,t} = INVH_{i,t} / STD(INVH_{i,t-3}, INVH_{i,t-2}, INVH_{i,t-1})（持股波动比指标），再以年-行业中位数为门槛，SD_{i,t} ≥ MEDIAN_{i,t}(SD_{i,t}) 记为稳定型 (INVW=1)，否则交易型 (INVW=0)。
  - [[short-selling-eligibility]] SSC：股票是否进入融资融券标的池。After：当期是否处在该股纳入两融之后的年度。
- 控制变量：Ret（公司年均特有周收益率）、Sigma（公司特有周收益率标准差）、Roa、Lev、Tobin's q、Dturn（趋势换手率）、Size（资产对数）、Accm（修正 Jones 模型分行业分年度估计的可操纵性应计利润）、年份与行业虚拟变量。

## 模型设定

H1 与 H2（OLS）：

```
Crash_{i,t} = α0 + α1 INVH_{i,t-1} + α2 Ret_{i,t-1} + α3 Sigma_{i,t-1}
            + α4 Roa_{i,t-1} + α5 Lev_{i,t-1} + α6 TobinQ_{i,t-1}
            + α7 Dturn_{i,t-1} + α8 Size_{i,t-1} + α9 Accm_{i,t-1}
            + α10 Year + α11 Ind + ε_{i,t-1}
```

H2 检验通过 INVW=1（稳定型）与 INVW=0（交易型）样本组分别回归（[[two-way-fixed-effects-industry-year]] 类似设计，但本文以行业/年度虚拟变量而非高维固定效应实现）。

H3（DID 三重交互）：

```
Crash_{i,t} = γ0 + γ1 INVH_{i,t-1}*SSC_{i,t-1}*After_{i,t-1}
            + γ2 INVH*SSC + γ3 SSC*After + γ4 INVH + γ5 SSC + γ6 INVW
            + 控制变量 + Year + Ind + ε
```

详见 [[did-short-selling-pilot]]。

## 主要结果

- 表 4（H1）：INVH 系数对 Ncskew 为 1.4319（t=10.33）、对 Duvol 为 0.9813（t=5.42），均在 1% 水平显著为正。机构投资者整体持股加大未来股价崩盘风险。
- 表 5（H2）：稳定型样本组（Panel A）中 INVH 系数为 −0.4714（Ncskew，5% 水平）和 −0.3261（Duvol，5% 水平）；交易型样本组（Panel B）中 INVH 系数为 1.5185（Ncskew，1%）和 0.2778（Duvol，5%）。两类机构投资者方向相反。
- 表 6（H3，DID）：稳定型样本三重交互项 INVH×SSC×After 系数 −0.7895（Ncskew, 1%）、−0.5771（Duvol, 1%）；交易型样本相应系数 −0.5680（Ncskew, 5%）、−0.6683（Duvol, 1%）。卖空机制松绑后稳定型机构投资者降低崩盘风险的作用增强，交易型机构投资者推升崩盘风险的作用减弱。

## 机制检验

本文未单独设置中介模型，理论机制通过两条渠道说明：

- [[stable-investor-monitoring-channel]]：稳定型机构投资者持股期长、持股比例高，监督上市公司、压缩坏消息囤积空间，从而降低崩盘风险。
- 卖空机制定价效率渠道（与本文 H3 紧耦合，未单立 mechanism 页）：松绑后悲观私有信息可融入股价，使股票更接近基本面；同时为稳定型机构投资者提供新治理工具，约束交易型机构投资者推升股价的能力。

## 异质性检验

按 [[institutional-investor-stability-split]] 分组：稳定型 (INVW=1) vs 交易型 (INVW=0)。两组样本量分别为 2128 和 2036（DID 时再按 SSC 进一步约束至 2020 / 1669）。结论：两类机构投资者对股价崩盘风险方向相反；卖空机制对两类机构投资者均产生抑制崩盘的影响，且对稳定型样本影响更显著。

本文还在引言与贡献部分明确剔除创业板和中小板，与曹丰等 (2015) 的全 A 股结论形成异质性对比。

## 稳健性检验

- 替代被解释变量 + Logistic 估计：以 IsCrash（W_{i,t} ≤ Average(W_{i,t}) − 3.09 σ_{i,t} 的事件型 0/1 哑变量）替代 Ncskew/Duvol，对模型 (6) 和 (7) 重新做 Logistic 回归（表 7、表 8）。INVH 系数对全样本和交易型样本组显著为正，对稳定型样本组显著为负；DID 三重交互项在两组样本中均在 1%/10% 水平显著为负，结论与基准一致。

## 内生性处理

- [[did-short-selling-pilot]]：以 2010 年 3 月 31 日融资融券试点为外生冲击，采用 DID 设计三重交互项 INVH×SSC×After，借助标的扩容外生差异识别卖空机制对机构投资者-崩盘风险关系的影响。
- 主回归中所有解释变量与控制变量均滞后一期，缓解反向因果。
- 行业 + 年份虚拟变量吸收宏观与行业层面共同冲击。

## 可复现线索

- 数据库表：[[csmar]] 上市公司股票交易周收益、流通市值加权市场周收益、机构投资者持股、财务报表（资产、负债、净利润、收入、应收账款、固定资产）、融资融券标的目录。
- 关键 Stata 步骤：
  - `merge` 周收益与市场指数；按 firm-year 跑 Dimson 5 项回归取残差，构造 W_{i,t}。
  - 按 firm-year 聚合 W 计算 Ncskew（公式 2）、Duvol（公式 3）、IsCrash（3.09σ 阈值）。
  - 计算 INVH、SD = INVH / rolling 3-year sd(INVH)、年-行业 INVW 虚拟变量。
  - 构造 SSC（融资融券标的虚拟）与 After（年度后 dummy）。
  - 构造 Accm：分年度分行业修正 Jones 模型残差。
  - OLS / 子样本 OLS / Logistic 回归，行业和年份虚拟变量。

## 对我当前选题的启发

- 这篇是耐心资本文献综述里的 B2 框架代表：以"持股波动率比 SD = INVH / 三年 INVH 标准差"为核心，配合行业-年中位数门槛把异质机构投资者二值化为稳定型/交易型。这个口径和 [[patient-capital]] 的"股权侧 = 稳定型机构投资者"操作化思路完全吻合，可以作为 B2 框架在崩盘风险被解释变量上的范式样本。
- 与 A2 框架（代飞 2025 类的 standard-deviation 比值连续测度）相比，B2 框架进一步把连续 SD 二值化、去掉绝对水平，便于异质性分组而牺牲了回归中的信息量；做项目时可对照两种口径给出敏感性。
- DID 设计（融资融券外生冲击）是耐心资本/机构投资者文献中处理内生性的标准模板之一。
- 文章对崩盘风险的两个测度（Ncskew、Duvol）和稳健性变量（IsCrash）是后续股价崩盘文献的标准三件套。
