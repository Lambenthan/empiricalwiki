## 研究问题

耐心资本能否提升企业全要素生产率（TFP）？该过程是否遵循边际报酬递减规律？背后的微观传导机制是创新效率与不确定性感知吗？传统创新效率与颠覆式创新效率（人工智能）对该作用是强化还是削弱？哪些因素制约了耐心资本的培育？

## 理论机制

作者基于内生增长理论与现代公司金融理论，构建一个包含耐心资本 K_p、传统资本 K_n 和劳动 L 的 Cobb-Douglas 内生增长模型 Y = A·K_p^α·K_n^β·L^(1-α-β)，并把 TFP 写成 A = exp(γI − δU)，其中 I = 创新效率、U = 不确定性感知，且 I = ΦK_p^θ。在 0 < θ < 1 时对 K_p 求二阶偏导为负，从而推出"耐心资本提升 TFP，且边际报酬递减"的核心命题。

机制分两条：

- 创新效率渠道：耐心资本以长期、稳定的资金缓冲企业短期财务压力，提高高风险研发承担能力，转化为更高的专利产出与创新产出比；颠覆式（AI）创新与耐心资本战略契合度更高，会强化主效应；传统创新由于资源向低风险短回报项目倾斜，反而削弱主效应。
- 不确定性感知渠道：耐心资本通过持续资本投入对冲外部宏观、市场、技术、地缘政治等不确定性，降低管理层的"不确定性感知"，避免推迟长周期项目，进而提高决策质量与生产效率。

进一步推论：风险承担水平是耐心资本培育的制约——企业风险承担过高时，投资者向其提供耐心资本的动机被削弱，进而部分阻断上述两条渠道。

## 研究假设

- H1：耐心资本能够提升企业 TFP，且该过程遵循边际报酬递减规律。
- H2：耐心资本通过提高企业创新效率，提升 TFP。
- H3：耐心资本通过降低企业不确定性感知，提升 TFP。

## 数据与样本

- 样本区间：2009—2022 年（基准回归口径，原文摘要写 2009–2022；正文表述与 sample_period 字段保持一致）。
- 样本范围：沪深 A 股上市公司。
- 样本筛选：剔除资产负债率 > 1、ST 类、金融保险行业；控制变量整体滞后一期。
- 样本规模：基准回归 33480 个企业-年观测（部分模型因加入个体固定效应或交互项掉到 27000–32964）。
- 数据来源：[[csmar]]（财务、治理、机构投资者持股、研发投入），上市公司年报 MD&A 文本（用于不确定性感知词频与人工智能词典）。

## 变量设定

- 被解释变量：[[total-factor-productivity]] (TFP) — 基准回归用 LP 法估计，稳健性中替换为 OLS 法（TFP_OLS）和 OP 法（TFP_OP）。基于 Cobb-Douglas 生产函数的两阶段估计：第一阶段利用中间投入估计生产率，第二阶段回归确定生产函数参数。

- 核心解释变量：[[patient-capital]] (Pat) — 本文采用 **C 框架（综合指标 / 熵值法）**：从耐心资本"长期导向、风险承受力、战略性关系性"3 个一级维度构造 7 个二级指标，标准化后用客观熵权法合成综合得分（详见下方"耐心资本变体说明"小节）。

- 中介变量：
  - [[innovation-efficiency]] (Ineff) — 三种专利（发明、实用新型、外观设计）数量加 1 取自然对数得到专利申请数，再用单位研发投入转化的专利申请数刻画创新效率（参考虞义华等 2018、方先明和胡丁 2023）。
  - [[uncertainty-perception]] (Uncer) — 上市公司年报 MD&A 部分不确定性相关词汇出现频次 / 该部分总词频（参考聂辉华等 2020、Yu et al. 2021）。

- 调节变量：
  - 传统创新效率 Tr_In — 企业发明专利申请数 + 1 的对数（易靖韬等 2015）。
  - 颠覆式创新效率 AI_In — 用机器学习生成 AI 词典并构建企业人工智能发展水平指标（姚加权等 2024）。

- 控制变量：金融化水平 Fin、上市年限 Age、总资产报酬率 Roa、营业收入增长率 Growth、第一大股东持股比例 Top1，全部滞后一期。

- 进一步分析：风险承担水平 Risk — 借鉴胡浩然和宋颜群（2022）。

### 耐心资本变体说明（C 框架 / 熵值法）

| 一级指标 | 二级指标 | 计算方法 | 方向 | 权重 |
|----------|----------|----------|------|------|
| 长期导向 | 投资者长期持股比例 | Yan and Zhang (2009)、黎文靖和路晓燕 (2015) | + | 14.6 |
| 长期导向 | 投资者短期持股比例 | Yan and Zhang (2009)、黎文靖和路晓燕 (2015) | − | 12.4 |
| 长期导向 | 投资者持股比例 / 过去 3 年持股比例标准差 | 吴旻佳等 (2022) | + | 10.1 |
| 风险承受力 | 长期资本负债率 | 非流动负债 / (所有者权益 + 非流动负债) | + | 17.9 |
| 风险承受力 | 企业短期财务杠杆 | 短期借款 / 权益账面价值 | − | 18.7 |
| 战略性关系性 | 资本增长保持率 | 本年所有者权益 / 上年所有者权益 | + | 16.6 |
| 战略性关系性 | 短视主义 | 胡楠等 (2021) | − | 9.7 |

操作化要点：先把 7 个二级指标标准化（保留方向），再用客观熵权法计算各指标信息熵 → 差异系数 → 权重，最终加权合成 Pat 综合得分。稳健性中作者把熵权法改为"各指标加总取平均"的简单加总作为稳健性。

样本均值 Pat = 0.032、标准差 0.030（最大 0.527），数值远小于 A 框架（量纲为比值之和）。

## 模型设定

主回归 → [[two-way-fixed-effects-industry-year]]（本文采用"个体 + 时间"双向固定效应，与既有页面同型）：

- 模型 (9) 一次项：TFP_it = β₀ + β₁·Pat_it + γ·X_(it−1) + Id_i + Year_t + ε_it
- 模型 (10) 边际报酬递减：TFP_it = β₀ + β₁·Pat_it² + β₂·Pat_it + γ·X_(it−1) + Id_i + Year_t + ε_it
- 模型 (11)–(13) 中介效应（M ∈ {Ineff, Uncer}）：
  - Pat → M：M_it = η₀ + η₁·Pat_it + γ·X'_(it−1) + Id_i + Year_t + ε_it
  - M → TFP：TFP_it = η₀ + η₁·M_it + γ·X_(it−1) + Id_i + Year_t + ε_it
  - Pat + M → TFP：TFP_it = η₀ + η₁·Pat_it + η₂·M_it + γ·X_(it−1) + Id_i + Year_t + ε_it
- 模型 (14) 调节效应（D ∈ {Tr_In, AI_In}）：TFP_it = α₀ + α₁·Pat_it + α₂·D_it + α₃·Pat_it × D_it + γ·X_(it−1) + Id_i + Year_t + ε_it

标准误：怀特异方差稳健，企业层面聚类（稳健性中替换为行业、地区聚类）。

## 主要结果

- 基准回归（表 3 列 4）：Pat 系数 1.827***（t = 6.56），调整 R² = 0.854；列 5 加入 Pat²，β₁(Pat²) = −5.061***、β₂(Pat) = 2.276***，验证耐心资本提升 TFP 且边际报酬递减（**H1 成立**）。
- 改变 PC 衡量方式（表 4 列 3）：用简单加总平均替代熵值法，Pat 系数仍显著为正（4.28e-11***，t = 2.76），结论稳健。
- 改变 TFP 衡量方式：TFP_OLS（系数 2.755***）、TFP_OP（系数 1.281***）均 1% 显著为正。
- 改变聚类层级（行业、地区）、上下 1%/2%/5% 缩尾、剔除 2020–2022 疫情样本、剔除四个直辖市样本，Pat 均在 1% 水平显著为正。
- IV-2SLS（表 6）：
  - IV1 = Pat 滞后一期：第一阶段 0.697***（t = 149.82），第二阶段 Pat 系数 15.879***（z = 38.65），弱工具变量 F = 22446.7。
  - IV2 = (1/到上海、深圳的加权距离) × 滞后一期上市年限：第一阶段 0.518***（t = 6.09），第二阶段 Pat 系数 311.36***（z = 6.29），F = 37.13；过度识别检验通过。

## 机制检验

[[innovation-efficiency-uncertainty-mediation]]：

- 表 7 列 (1)：Pat → Ineff 系数 0.159***（t = 7.92），耐心资本促进创新效率。
- 表 7 列 (2)：Ineff → TFP 系数 0.609***（t = 6.76）。
- 表 7 列 (3)：同时纳入 Pat 与 Ineff，Pat 系数从 1.827 降至 1.218***，Ineff 系数 0.589***，部分中介。
- 表 7 列 (4)：Pat → Uncer 系数 −0.144***（t = −3.99），耐心资本降低不确定性感知。
- 表 7 列 (5)：Uncer → TFP 系数 −0.231***。
- 表 7 列 (6)：同时纳入，Pat 系数 1.796***，Uncer 系数 −0.221***。
- Bootstrap 1000 次抽样：Ineff 路径 95% CI [1.73386, 2.09952]，Uncer 路径 95% CI [0.08903, 0.20104]，均不含 0；Sobel 统计量分别 22.46***、5.55***。**H2、H3 成立**。

调节效应（表 8）：

- Pat × Tr_In = −0.429***（t = −4.06），传统创新效率削弱主效应。
- Pat × AI_In = 1.172**（t = 2.36），颠覆式创新（AI）强化主效应。

## 异质性检验

- 企业性质（表 9）：国有企业子样 Pat = 2.133***、Pat² = −5.091***，倒 U 拐点 0.264；非国有企业 Pat = 1.601***，Pat² 不显著。bdiff（1000 次）p = 0.000，国企的 Pat 系数显著高于非国企。
- 行业金融化水平（表 10）：以行业内年度金融化中位数划分。金融化较低样本 Pat = 1.849***（Pat² = −4.622***），金融化较高样本 Pat = 1.635***（Pat² = −11.65***）。bdiff p = 0.029、0.052，金融化较低行业的促进作用更大。

## 稳健性检验

- 改变 TFP 衡量方式：基准 LP 法 → OLS 法 / OP 法（表 4 列 1–2）。
- 改变 Pat 衡量方式：熵值法 → 简单加总平均（表 4 列 3）。
- 改变聚类层级：企业 → 行业、地区（表 4 列 4–5）。
- 缩尾稳健性：上下 1% / 2% / 5% 缩尾（表 5 列 1–3）。
- 排除特殊时间：剔除 2020—2022 年新冠疫情样本（表 5 列 4）。
- 排除特殊地区：剔除四个直辖市样本（表 5 列 5）。

## 内生性处理

主识别策略 = 双工具 2SLS（[[instrumental-variable-2sls]]）：

- IV1（时间维度）：Pat 滞后一期。
- IV2（空间维度）：[[geographic-distance-iv]] — 各企业到上海、深圳两个金融中心加权距离的倒数 × 滞后一期上市年限。逻辑：地理位置外生于 TFP，但越靠近金融中心越易获得金融服务、越易形成耐心资本。

辅助识别：个体（企业）固定效应 [[firm-fixed-effects]]，控制企业层面不可观测的时不变异质性；控制变量整体滞后一期，缓解反向因果。

## 可复现线索

- 解释变量构造：先准备 7 个二级指标（CSMAR 机构投资者持股表、长期借款明细、所有者权益、短期借款、年报 MD&A 文本短视词典），按方向标准化（正向 (x − min)/(max − min)，负向 (max − x)/(max − min)），再 Stata `entropy` 命令或自写信息熵代码计算权重，加权合成 Pat。
- 被解释变量构造：LP 法可用 `levpet` / `prodest` / `xtfrontier` 等命令；变量需对数化的 Y、L、K、M（增加值或营业收入、员工人数、固定资产、中间投入）。OP 法用 `opreg`。
- 创新效率：CSMAR 研发投入 + CNRDS / CPDP 专利数据库；Ineff = ln(发明 + 实用新型 + 外观设计 + 1) / 研发投入。
- 不确定性感知：从巨潮资讯网下载年报 PDF → 抽 MD&A 段 → jieba 分词 → 词典法统计不确定性词频 / 该部分总词频。
- AI 调节变量：参照姚加权等 (2024) 的 AI 词典 + 年报词频。
- Stata 命令：`reghdfe TFP Pat L.Fin L.Age L.Roa L.Growth L.Top1, absorb(firmid year) cluster(firmid)` 复现表 3 列 4；IV 用 `ivreghdfe TFP (Pat = L.Pat IV2) L.controls, absorb(firmid year) cluster(firmid)`。
- bdiff 系数差异检验：可用 `suest` + `test` 或 `bsample` 自助法 1000 次。

## 对我当前选题的启发

- 本文是耐心资本测度的 **C 框架范本**：用熵值法把"长期导向 / 风险承受力 / 战略性关系性"三维 7 项指标合成单一指标，相对 A 框架（关系型债务 + 稳定型股权之和）更全面地纳入了"短视主义"等行为变量。本项目可在主分析中并列报告 A、B、C 三类 PC 测算，互为稳健性。
- 全要素生产率 TFP 的多种估计法（LP 主、OLS / OP 副）是中文实证 TFP 类论文的标配；可作为本项目的 TFP 测度模板。
- 创新效率 Ineff（专利 / 研发）+ 不确定性感知 Uncer（MD&A 词频）这组双中介可与 [[managerial-myopia-mediation]] 形成对照：本文走的是"创新供给 + 决策稳定性"渠道，代飞 (2025) 走的是"管理者短视"渠道，二者并非互斥。
- 颠覆式创新（AI 词频）作为调节变量的做法可直接迁移到本项目作为耐心资本影响 TFP 的边际改良工具。
- IV2（金融中心地理距离）是相对新颖的空间工具变量，但需注意作者表 6 中第二阶段 Pat 系数（15.879、311.36）与基准回归的 1.827 数值差距巨大，原因可能是 2SLS 把局部平均处理效应放大了，使用时需谨慎解释。
