## 研究问题

耐心资本（含稳定型股权与关系型债权两个维度）能否提升沪深 A 股制造业企业的新质生产力？营商环境是否对这一提升效应起正向调节作用？耐心资本影响新质生产力的中介机制是融资约束缓解还是企业创新水平提升？该效应在地区属性、内部控制质量与信息不对称程度不同的制造业企业之间是否存在异质性？

## 理论机制

杨芳 (2025) 构建"耐心资本 → 新质生产力"主链路，并在外部条件层引入营商环境调节。

主链路（图 1 机制路线）：

- 耐心资本承受经济短期波动压力（Ching, 2021），发挥市场稳定器作用。
- 反馈效应：耐心资本让股价反馈效应外显（陆蓉、孙欣钰, 2021），强化市场信息在企业创新中的引导作用，弥补管理层外部信息不足。
- 长期价值评估：耐心资本以更长远战略作为评价标准（姜中裕、吴福象, 2024），筛选具长期发展潜力的优质项目，更重视技术创新能力与市场增长潜力等非财务指标。
- 多元化投资主体：政府基金、企业私人投资者、金融机构等共同构建韧性创新网络平台，整合协同资源、提升抗风险能力。

中介路径分两条平行子机制：

- 融资约束中介（H2 → [[financing-constraint-mediation]]）：稳定型股权通过参与公司治理、均衡企业内外创新信息差、声誉机制三重作用降低信息不对称、缓解融资约束；关系型债权通过广泛信息渠道与挖掘能力、保护性条款限制资金错配、多元投资主体战略联盟规模效应缓解约束。
- 企业创新中介（H3 → [[innovation-mediation-joint-patents]]）：稳定型股权发挥信息优势、监督优势、战略优势；关系型债权提供资金支持（信誉评级机构担保）、信息支持（长期合作信任）、机制支持（商业银行固定收益索取权"挤出"经理人私利、风险对冲机制提升创新风险包容性）。

调节路径（H4 → 营商环境调节）：

- 资源配置：良好营商环境下创新资源持续汇聚，市场资源配置作用增强，避免因资金短缺导致创新中断。
- 投资文化：政府与市场关系顺畅、企业融资约束较小，社会投资者动机增强（牛鹏等, 2022），打造长期投资文化。
- 法律保障：政策法规越完善，耐心资本稳定性与可预测性越高，激发企业创新活动；创新成果商业化门槛降低、市场应用效率提高，形成"虹吸效应"吸引更多耐心资本。

## 研究假设

- H1：耐心资本能够显著提升制造业企业新质生产力。
- H2：耐心资本通过缓解融资约束，进而提升制造业企业新质生产力。
- H3：耐心资本通过提高企业创新，进而提升制造业企业新质生产力。
- H4：营商环境的优化能够正向调节耐心资本对制造业企业新质生产力的提升作用。

## 数据与样本

- 样本区间：2011—2022 年。
- 样本范围：沪深 A 股制造业上市公司。
- 样本筛选：剔除数据严重缺失样本、缺乏重要财务指标样本、ST 或 *ST 等经营异常样本；所有变量上下 1% 缩尾。
- 最终样本：12230 个企业-年度观测（基准回归因控制变量缺失降至 N = 10119）。
- 数据来源：
  - 机构投资者数据 → [[csmar]]
  - 研发相关数据 → [[wind]]
  - 财务数据（长期负债、负债总额、财务信息、股票信息、企业性质）→ [[csmar]]
  - 营商环境数据 → 《中国省份营商环境研究报告》（缺失年份线性插值）

## 变量设定

- 被解释变量：[[new-quality-productive-forces]] (Npro)
  - 借鉴宋佳等 (2024)，根据生产力二因素理论，以活劳动、物化劳动（劳动对象）、硬科技、软科技四大子因素为基础构建评价体系（共 11 个底层指标，权重合计 100；详见原文表 1）。
  - 子指标包括：研发人员薪资占比、研发人员占比、高学历人员占比、固定资产占比、制造费用占比、研发折旧摊销占比、研发租赁费占比、研发直接投入占比、无形资产占比、总资产周转率、权益乘数倒数。
  - 合成方法：熵值法计算权重并合成综合得分。
  - 描述性统计（表 3）：均值 5.0972、中位数 4.8499、SD 2.0820、最大 27.0411、最小 0.392。
- 核心解释变量：[[patient-capital]] (Pa)，分两个维度（A1 经典版口径）：
  - 稳定型股权 [[stable-institutional-investors-turnover]] (Wd) — 借鉴周绍妮等 (2017)，按机构投资者换手率三分组，最低组为稳定型机构投资者，按公司汇总持股比例。描述统计：均值 0.0129、中位数 0.0029、SD 0.0237、最大 0.3868、最小 0.0000。
  - 关系型债权 [[relational-debt-total-long-debt-ratio]] (Pc) — 借鉴姜中裕、吴福象 (2024)，长期负债 / 负债总额。描述统计：均值 0.1305、中位数 0.0809、SD 0.1452、最大 0.8701、最小 0.0000。
- 中介变量：
  - 融资约束 (WW) — 借鉴刘莉亚等 (2015) 构建的 WW 指数；数值越大融资约束越高。
  - 企业创新 (EI) — 制造业企业当年联合申请专利总数。
- 调节变量：[[business-environment-china-province-report]] (BE) — 借鉴白钰、冯均科 (2024)，《中国省份营商环境研究报告》指数 + 线性插值，按省份-年份匹配企业。
- 控制变量（参考姜中裕、吴福象 2024；吴旻佳等 2022；杨芳等 2024）：
  - ROA = 净利润 / 总资产
  - FirmAge = Ln(成立年数)
  - Opinion = 标准审计意见哑变量（标准 = 1）
  - Top1 = 第一大股东持股比例
  - Dual = 两职合一哑变量
  - Cashflow = 年末经营性现金流量 / 主营业务收入
  - SOE = 国有企业哑变量（国企 = 1）
  - Growth = 营业收入增长率
  - Indep = 独立董事 / 董事会人数
  - Board = Ln(董事会人数)
  - Mshare = 管理层持股 / 总股数
- 固定效应：行业（Ind）+ 年份（Year），模型设定中固定效应均控制。

## 模型设定

主回归 → [[two-way-fixed-effects-industry-year]]：

- 模型 (1) 基准：Npro(i,t) = β0 + β1·Pa(i,t-1) + Controls(i,t-1) + Year(i,t-1) + Ind(i,t-1) + ε(i,t-1)
- 模型 (2) 机制（江艇 2022 路径）：Medium(i,t) = β0 + β1·Pa(i,t-1) + Controls(i,t-1) + Year(i,t-1) + Ind(i,t-1) + ε(i,t-1)，Medium ∈ {WW, EI}
- 模型 (3) 调节效应：Npro(i,t) = β0 + β1·Pa(i,t-1) + β2·BE(i,t-1) + β3·Pa(i,t-1)×BE(i,t-1) + Controls(i,t-1) + Year(i,t-1) + Ind(i,t-1) + ε(i,t-1)

注：所有解释变量与控制变量均滞后一期处理以缓解内生性问题；Pa 在所有模型中分别用 Wd 与 Pc 替换，得到两套平行结果。

## 主要结果

- 描述性统计（表 3）：N = 12230；Npro 均值 5.0972、SD 2.0820；Wd 均值 0.0129、SD 0.0237；Pc 均值 0.1305、SD 0.1452。耐心资本两个维度在样本企业间存在显著差异。
- 基准回归（表 4，N = 10119）：
  - 列 (1) 不加控制变量、不加 FE：Wd 系数 8.3889***（t = 9.5894）。
  - 列 (2) 不加控制变量、不加 FE：Pc 系数 1.6684***（t = 11.7436）。
  - 列 (3) 不加控制变量、加 Ind 与 Year FE：Wd 系数 8.5591***（t = 9.9530）。
  - 列 (4) 不加控制变量、加 Ind 与 Year FE：Pc 系数 1.4807***（t = 10.6106）。
  - 列 (5) 加控制变量、不加 FE：Wd 系数 7.5891***（t = 8.6947）。
  - 列 (6) 加控制变量、不加 FE：Pc 系数 1.4300***（t = 10.1325）。
  - 列 (7) 加控制变量、加 FE（最终设定）：Wd 系数 7.8067***（t = 9.1276），adj. R² = 0.0753。
  - 列 (8) 加控制变量、加 FE（最终设定）：Pc 系数 1.3208***（t = 9.5655），adj. R² = 0.0923。
  - H1 通过：耐心资本显著提升制造业企业新质生产力。
- 调节效应（表 5 列 5-6）：
  - 列 (5) c.Wd × c.BE 系数 2.2057***（t = 9.3451），adj. R² = 0.0761。
  - 列 (6) c.Pc × c.BE 系数 0.3597***（t = 9.2721），adj. R² = 0.0922。
  - H4 通过：营商环境的优化正向调节耐心资本对新质生产力的提升作用，无论从股权侧还是债权侧均成立。
- 控制变量（表 4 列 7-8 节选）：ROA、Cashflow、Growth、Board、Top1、SOE、FirmAge、Opinion 大多在 1% 或 5% 水平显著；Dual 与 Mshare 系数较小或不显著。

## 机制检验

[[financing-constraint-mediation]] 与 [[innovation-mediation-joint-patents]] 两条平行路径，借鉴江艇 (2022) 机制检验思路：

- 表 5 列 (1) Wd → WW：系数 -0.4651***（t = -20.1490），Sobel p = 0.0372 < 0.05；adj. R² = 0.3631。
- 表 5 列 (2) Pc → WW：系数 -0.0657***（t = -17.4985），Sobel p = 0.0091 < 0.05；adj. R² = 0.3560。
- 表 5 列 (3) Wd → EI：系数 247.9703***（t = 2.7334），Sobel p = 0.0346 < 0.05；adj. R² = 0.0052。
- 表 5 列 (4) Pc → EI：系数 55.7405***（t = 3.3916），Sobel p = 0.0137 < 0.05；adj. R² = 0.0060。

H2 与 H3 均通过：耐心资本通过缓解融资约束（系数为负且显著）与提高企业创新水平（系数为正且显著）双路径提升制造业企业新质生产力。两路径互补，不是替代关系。

> 注：表 5 列 3-4 的 R² 极低（0.005），提示企业创新（联合申请专利绝对数）残差较大，结论稳健性需配合 Sobel 检验。

## 异质性检验

杨芳 (2025) 采用"交乘项"路径而非分组分别回归（表 7）。三个维度异质性：

1. 地区属性（[[eastern-region-split]]）：
   - East = 1 东部地区、East = 0 中西部地区。
   - 列 (1) c.Wd × c.East 系数 7.5219***（t = 7.6627）；列 (2) c.Pc × c.East 系数 1.1551***（t = 7.5108）。
   - 东部地区耐心资本提升新质生产力的作用更强。

2. 内部控制：
   - 用迪博内部控制信息披露指数（DIB IC index）测度，数值越大内控水平越高。
   - 列 (3) c.Wd × c.IC 系数 0.2126***（t = 9.4948）；列 (4) c.Pc × c.IC 系数 0.0360***（t = 9.6728）。
   - 内控质量较高的制造业企业，耐心资本对新质生产力的提升效应更强。

3. 信息不对称：
   - 参考宋敏等 (2021) 与于蔚等 (2012) 的方法测度，得到信息不对称综合指标 Asy。
   - 列 (5) c.Wd × c.Asy 系数 -8.8551***（t = -11.4831）；列 (6) c.Pc × c.Asy 系数 -1.3271***（t = -10.2285）。
   - 信息不对称程度较低（Asy 较小）的企业，耐心资本对新质生产力的提升效应更强（交乘项符号为负，表示 Asy 增大时主效应减弱）。

> 注：内部控制与信息不对称两个异质性切分本论文以"交乘项 + 全样本回归"形式实现，不构造单独子样本，因此本 wiki 仅为地区属性新建独立 [[eastern-region-split]] 页面，IC 与 Asy 异质性以本卡片描述方式记录。

## 稳健性检验

1. **Heckman 两阶段（IV）**（表 6 列 1-4）：
   - 工具变量：上市公司同年同行业剔除自身企业的耐心资本均值（ivmeanWd / ivmeanPc）。
   - 列 (1) 第一阶段 ivmeanWd → Wd 系数 0.8073***（t = 11.7810），F = 67 > 10，排除弱工具问题。
   - 列 (2) 第二阶段 Wd → Npro 系数 62.9137***（t = 6.9067）。
   - 列 (3) 第一阶段 ivmeanPc → Pc 系数 0.9020***（t = 13.6660），F = 38 > 10。
   - 列 (4) 第二阶段 Pc → Npro 系数 15.1677***（t = 10.4189）。
   - IV 估计与基准方向一致，量级显著放大（IV LATE 通常大于 OLS 平均效应）。
2. **安慰剂检验**（表 6 列 5-6）：
   - 将耐心资本随机分配给样本企业，重复回归 2000 次。
   - 列 (5) Wd 安慰剂系数 -0.0215（t = -0.5485）不显著；列 (6) Pc 安慰剂系数 -0.0204（t = -0.5745）不显著。
   - 系数与 p 值分布（图 2、图 3）显示 p 值多大于 0.1，证明 Npro 提升确实由耐心资本引起，结论稳健。
3. **倾向得分匹配（PSM）**（表 6 列 7-8）：
   - Wd：以均值为标准分组（高于均值 = 实验组、低于 = 对照组），1:1 有放回匹配得 5913 个样本。匹配后 Wd 系数 0.4574***（t = 7.8026）。
   - Pc：同方法匹配得 7577 个样本。匹配后 Pc 系数 0.2978***（t = 6.0755）。
   - 控制组间差异后耐心资本系数仍显著为正，匹配效果较好（图 4、图 5 倾向得分集中在 0 附近）。

## 内生性处理

主要识别策略采用 Heckman 两阶段法 + IV 同行业同年份剔除自身均值（[[iv-industry-mean-pc-excluding-self]] 框架的同类应用）：

- 相关性：行业内同行其他企业的耐心资本水平与本企业耐心资本高度相关（同行竞争压力 + 区域要素溢出）；第一阶段 F 值 67（Wd）/ 38（Pc）均大于 10，排除弱工具问题。
- 排他性：本企业新质生产力受企业内部条件（研发、人才、固定资产）主导，行业其他企业的耐心资本不构成对本企业 Npro 的直接通道。
- 第二阶段系数（62.9137 / 15.1677）方向与基准一致，量级放大约 8-11 倍，符合 IV 局部平均处理效应（LATE）特征。

辅助处理：

- 全部主解释变量与控制变量滞后一期（缓解反向因果）。
- 安慰剂检验（2000 次随机分配处理变量，验证非伪回归）。
- PSM 1:1 有放回匹配（缓解处理组与对照组特征差异导致的选择偏误）。

## 可复现线索

- **被解释变量 Npro**：从 [[csmar]] 取财务数据（研发费用、薪酬、员工人数、固定资产、无形资产、营业收入、资产总额、所有者权益）；从 [[wind]] 取研发明细（研发费用-工资薪酬、研发费用-折旧摊销、研发费用-租赁费、研发费用-直接投入）；通过员工统计取本科以上学历人数与研发人员数。11 个底层指标分别标准化后熵权法合成。
- **稳定型股权 Wd**：从 [[csmar]] 机构投资者持股表与机构投资者交易明细，按周绍妮等 (2017) 范式计算每个机构的研究期均值换手率（双向最小买卖额比，半年度数据，5 年研究期取均值），按 AVR_CR 等距三分组，最低组为稳定型；按公司年度合计稳定型机构投资者持股比例。
- **关系型债权 Pc**：从 [[csmar]] 财务报表合并口径资产负债表，Pc = 长期负债合计 / 负债合计。
- **融资约束 WW**：借鉴刘莉亚等 (2015) 的 WW 指数公式（CF、DIVPOS、TLTD、LNTA、ISG、SG 加权合成），从 [[csmar]] 现金流量表、利润表、资产负债表取相应字段。
- **企业创新 EI**：制造业企业当年联合申请专利总数（CNRDS 或 CNIPA 专利数据库 + 申请人字段判定联合申请）。
- **营商环境 BE**：《中国省份营商环境研究报告》各年度版本，缺失年份线性插值，按企业注册省份-年份合并。

* sobel 检验：sgmediation 或 medsem 或手算

* IV（表 6 列 1-4）

## 对我当前选题的启发

- 本论文采用 PC A1 经典口径（换手率三分组 + 总长债占比），与代飞 (2025) A2（标准差 + 银行长债）、简冠群 (2025) A3（标准差 + 非流动负债明细）形成完整 A 系列对比。本项目复现 PC 时可同时跑三个口径，比较系数的稳定性，作为测度敏感性分析。
- 营商环境调节效应是本论文核心创新点。BE 数据源（《中国省份营商环境研究报告》）与谢婷婷 (2025) 的王小鲁市场化指数完全不同，建议本项目在做调节效应稳健性时同时跑两套 BE 测度。
- 制造业子样本设计：单纯以制造业为研究对象（剔除金融、地产、服务业）能捕捉新质生产力的核心载体行业，但损失了行业层异质性。本项目"耐心资本 × 新质生产力"主线可考虑制造业 vs 全样本 vs 高技术制造业的样本敏感性。
- 异质性"交乘项 + 全样本"方法论：相比传统"分组分别回归 + 系数差异检验"，本论文的交乘项方法更具效率（保留全样本量、降低 SE），但要求三向交乘项的解释慎重（双重交乘可能让边际效应解读复杂）。
- 中介机制双路径并行：融资约束 + 企业创新两条独立中介，未做联合中介模型（VAR 或结构方程模型）识别相对重要性。本项目可补充 Bootstrap 中介检验或 PROCESS 宏验证两条路径的强弱顺序。
- 表 5 列 3-4 R² 极低（0.005）提示企业创新中介的统计推断脆弱，本项目复现时建议同时报告 Bootstrap CI 与 KHB 分解，避免依赖 Sobel p 值的单一推断。
- 内生性"IV + 滞后 + 安慰剂 + PSM" 四件套是本项目主分析的标准稳健性矩阵，可整套迁移；若样本期足够长，建议补充 GMM（参考简冠群 2025）。
