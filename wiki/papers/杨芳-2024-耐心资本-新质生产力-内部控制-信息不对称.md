## 研究问题

耐心资本（拆解为稳定型股权与关系型债权两个维度）的增加是否显著提升企业新质生产力发展水平？其作用机制是否通过提高内部控制质量与降低信息不对称程度两条路径实现？该影响是否在不同企业规模、股权集中度、市场竞争程度下表现出异质性？

## 理论机制

基于生产力三要素（劳动者、劳动资料、劳动对象）框架，本文提出耐心资本通过以下三条直接路径影响企业新质生产力：

第一条直接路径——劳动者素质提升：耐心资本的长周期成长性促使企业通过"干中学"提升员工技能；战略性思维投资者重视人力资本积累，加大对员工培训与人才引进的投入。

第二条直接路径——劳动资料改善：耐心资本注重风险规避与稳定投资导向，强化创新治理监督，通过长期资金支持推动生产技术与设备的改进升级，抑制为追求短期利润而忽视技术创新的行为。

第三条直接路径——劳动对象拓展升级：耐心资本通过推动技术研发投入、加速创新成果转化，引导企业使用更高质量、更环保的原材料以及新能源和可再生能源，开拓新市场。

机制层面，耐心资本兼具"稳定性"和"关系性"双重特征：稳定性来自投资者长期持有，使其具有动机参与企业治理；关系性来自投资者与企业建立的互信合作关系。这使得耐心资本能够分别通过：（1）改善内部控制质量来塑造企业内部环境（监督治理 + 可持续战略 + 降低代理成本）；（2）降低信息不对称来改善企业外部环境（信息获取优势 + 信誉传递 + 市场信息环境）来助力新质生产力发展。

## 研究假设

- H1：耐心资本（稳定型股权占比和关系型债权占比）的增加有助于企业新质生产力水平提升。
- H2：耐心资本的增加能够通过提高内部控制质量来促进企业新质生产力水平提升。
- H3：耐心资本的增加能够通过降低信息不对称程度来促进企业新质生产力水平提升。

## 数据与样本

- **样本区间**：2011—2022 年。
- **样本范围**：沪深 A 股上市公司。
- **样本筛选**：剔除数据严重缺失的样本、缺乏重要财务指标的样本、ST 或 *ST 等经营异常样本，最终获得 21430 个观测值；主回归因核心解释变量与控制变量滞后一期处理，有效观测降至 17182。
- **缩尾处理**：所有变量进行前后 1% 的缩尾处理。
- **数据来源**：机构投资者数据来自 [[csmar]]（国泰安数据库）；企业研发相关数据来自 [[wind]]（万得数据库）；企业财务数据来自 [[csmar]]。

## 变量设定

**被解释变量（Npro，新质生产力）**：借鉴张雪兰等（2024）方法，构建企业新质生产力指数评价指标体系。一级指标为新质劳动者（权重 0.276）、新质劳动资料（0.314）、新质劳动对象（0.193）、优化组合（0.217）；每个一级指标下设科技创新、未来产业发展、产业链布局 3 个二级指标；最终选取 27 个三级指标（如高学历员工占比、研发人员占比、研发薪酬占比、培训费用占比、数字化背景高管占比、研发费用占比、数字无形资产占比、人工智能采纳程度、数字技术创新、绿色技术创新、数据资产占比、产学研合作、企业 ESG 表现等）。采用熵值法分层赋权合成。详见 [[new-quality-productive-forces]]。描述统计：均值 13.1138，标准差 8.4732，最大值 72.0127，最小值 1.4323。

**核心解释变量（Pa，耐心资本）**：分解为两个维度：

1. **稳定型股权（Wd）**：借鉴周绍妮等（2017）方法，按机构投资者平均换手率将其划分为高、中、低 3 组，其中换手率最低组定义为稳定型机构投资者，进而计算每只股票中稳定型机构投资者持股比例之和。仅选取 2011—2022 年持股期为两期及以上的机构投资者。详见 [[stable-institutional-investors-turnover]]。描述统计：均值 0.0122，中位数 0.0025，标准差 0.0247，最大值 0.4360，最小值 0.0000。

2. **关系型债权（Pc）**：参考吴旻佳等（2022）、姜中裕和吴福象（2024）方法，使用上市公司长期负债（长期借款、应付债券、长期应付款等）占负债总额的比例衡量。详见 [[relational-debt-total-long-debt-ratio]]。描述统计：均值 0.1664，中位数 0.1038，标准差 0.1817，最大值 0.6273，最小值 0.0000。

**中介变量**：

- **内部控制（IC）**：采用迪博内部控制指数衡量。指数越大表明内部控制质量越高。
- **信息不对称（ASY）**：参照于蔚等（2012）、宋敏等（2021）方法，采用流动性比率、非流动性比率、收益率反转 3 个指标做主成分分析得到股票流动性指数（数值越大则信息不对称程度越高）。

**调节变量（异质性）**：

- 企业规模 = ln(总资产)。
- 股权集中度 = 第一大股东持股数 / 公司总股数。
- 市场竞争程度 = 行业赫芬达尔指数 HHI（数值越大则行业竞争程度越低）。

**控制变量（10 个，借鉴张雪兰等 2024、杨芳等 2024）**：资产收益率（净利润/总资产）、现金流比率（年末经营性现金流量/主营业务收入）、成长性（营业收入增长率）、董事会规模（董事会人数对数）、独立董事比例、两职合一（董事长兼总经理为 1）、第一大股东持股比例、企业年龄（成立年限对数）、管理层持股比例、审计意见（标准审计意见为 1）。

## 模型设定

基准模型（核心解释变量与控制变量均滞后一期）：

```
Npro_{i,t} = β₀ + β₁ Pa_{i,t-1} + β · Controls_{i,t-1} + Year_{i,t-1} + Ind_{i,t-1} + ε_{i,t-1}
```

其中 Pa 分别替换为"稳定型股权"和"关系型债权"两个变量分别回归，再合并加入。固定效应：行业固定效应 + 年份固定效应。详见 [[two-way-fixed-effects-industry-year]]。

中介模型（江艇 2022 两步法）：

```
Medium_{i,t} = β₀ + β₁ Pa_{i,t-1} + Controls_{i,t-1} + Year_{i,t-1} + Ind_{i,t-1} + ε_{i,t-1}
```

其中 Medium 分别替换为"内部控制"和"信息不对称"。

异质性（调节）模型：在基准模型中加入"调节变量 + 调节变量 × 稳定型股权 + 调节变量 × 关系型债权"交互项。

## 主要结果

基准回归结果（表 3，n = 17182）：

- 稳定型股权 → 新质生产力：未控制变量时系数 25.3525∗∗∗（t = 13.85）；加入控制变量后 22.3997∗∗∗（t = 12.01）。
- 关系型债权 → 新质生产力：未控制变量时系数 1.2356∗∗∗（t = 3.93）；加入控制变量后 1.1034∗∗∗（t = 3.41）。
- 调整 R² 介于 0.427 与 0.441 之间。

H1 得到验证；从系数大小看，"稳定型股权" 的系数明显大于 "关系型债权" 的系数，表明稳定型股权增加对新质生产力发展水平的提升作用更大。

## 机制检验

**内部控制路径（表 5 Panel A）**：

- 稳定型股权 → 内部控制：系数 7.9837∗∗∗（t = 5.23），Sobel p = 0.0093。
- 关系型债权 → 内部控制：系数 0.8041∗∗∗（t = 2.95），Sobel p = 0.0000。

**信息不对称路径（表 5 Panel B）**：

- 稳定型股权 → 信息不对称：系数 -3.5196∗∗∗（t = -29.23），Sobel p = 0.0000。
- 关系型债权 → 信息不对称：系数 -0.2804∗∗∗（t = -13.11），Sobel p = 0.0000。

H2、H3 均得到验证。详见 [[internal-control-channel]] 与 [[information-asymmetry-mediation]]。

## 异质性检验

**Panel A — 企业规模异质性（n = 17182）**：

- 稳定型股权 × 企业规模：0.9960∗∗∗（t = 12.63）。
- 关系型债权 × 企业规模：0.0567∗∗∗（t = 4.41）。

规模较大的企业耐心资本对新质生产力的促进作用更强。详见 [[heterogeneity-firm-size-moderator-pc-nqpf]]。

**Panel B — 股权集中度异质性**：

- 稳定型股权 × 股权集中度：0.7990∗∗∗（t = 3.97）。
- 关系型债权 × 股权集中度：0.0223∗∗∗（t = 3.77）。

股权集中度较高的企业耐心资本作用更强。详见 [[heterogeneity-equity-concentration-moderator-pc-nqpf]]。

**Panel C — 市场竞争程度异质性（HHI）**：

- 稳定型股权 × 市场竞争程度（HHI）：36.5379∗∗∗（t = 4.20）。
- 关系型债权 × 市场竞争程度（HHI）：2.3039∗（t = 1.96）。

行业竞争程度较低（HHI 较高）的企业耐心资本作用更强。详见 [[heterogeneity-market-competition-moderator-pc-nqpf]]。

## 稳健性检验

**Panel A — PSM 检验（表 4 Panel A）**：

- 对稳定型股权：按其均值划分实验组（高于均值）与对照组（低于均值），以基准控制变量为协变量，1∶1 有放回匹配。匹配后 n = 7836，PC 系数 2.3846∗∗∗（t = 14.97）。
- 对关系型债权：同样按均值划分组别，最近邻匹配。匹配后 n = 10358，PC 系数 1.1322∗∗∗（t = 8.07）。

**Panel B — 分段子样本检验**：

- 2011—2016 子样本（n = 5468/5467）：稳定型股权 5.5993∗∗（t = 1.95），关系型债权 0.1382∗（t = 1.95）。
- 2017—2022 子样本（n = 11746/9210）：稳定型股权 26.1733∗∗∗（t = 10.17），关系型债权 1.1620∗∗∗（t = 4.09）。

2017—2022 子样本系数显著性与绝对值均明显大于 2011—2016 子样本，说明耐心资本的新质生产力促进效应趋于增强。详见 [[robustness-sub-period-2011-2016-vs-2017-2022]]。

**Panel C — 安慰剂检验**：

将稳定型股权和关系型债权变量随机分配给样本企业构造反事实样本，重复 500 次。所得系数 p 值大多大于 0.1，表明基准结果并非由其他未观测因素引起。

## 内生性处理

主回归通过将核心解释变量与控制变量均滞后一期减轻同期反向因果。PSM 通过协变量平衡降低样本选择偏误；安慰剂检验排除遗漏变量驱动的可能。本文未使用工具变量方法或 DID 自然实验设计。

## 对我当前选题的启发

- 本论文是 A1 经典版（换手率 3 分组 + 总长债占比）框架的代表作之一，与 [[杨芳-2025-耐心资本-制造业-新质生产力-营商环境]]（同作者团队）形成了 PC → 新质生产力链路的"全样本"与"制造业"两个分支。
- 中介机制选择"内部控制 + 信息不对称"两条路径，与 [[李季鹏-2025-耐心资本-高质量发展]] 共享信息不对称机制（PCA 流动性指数测度），但用迪博 IC 而非治理指数代理内部控制，方法学上较成熟可复用。
- 异质性使用"调节变量 × PC"交互项形式（而非样本分组），适合本项目研究 PC 三种制度环境放大器（规模、股权集中度、HHI）。
- 稳健性检验组合（PSM + 分段子样本 + 安慰剂）较完整，可作为本项目稳健性最小套件参考。
- 可补充的不足：未使用 IV 方法处理 PC 内生性；未做 PC 异质性内部分组（如稳定型 vs 交易型 ratio）；未对新质生产力指标体系做敏感性分析（更换权重或子指标）。

## Variant Notes

**与 [[简冠群-2025-专精特新-新质生产力-风险投资-耐心资本]] 的差异（共享 NQPF 但 PC 测度框架不同）**：

- 因变量同为新质生产力（NQPF），但杨芳 (2024) 借鉴张雪兰等（2024）27 个三级指标 + 熵值法（一级指标四类：劳动者/劳动资料/劳动对象/优化组合），与简冠群 (2025) 的具体指标体系可能存在差异（需对照简冠群 paper 卡片确认权重与指标层级）。
- PC 测度：杨芳 (2024) 采用 A1 经典版（换手率 3 分组 + 长期负债/负债合计），简冠群 (2025) 采用 A3 框架（持股稳定性 + 长期负债明细的并行机制变量）。两者在债权侧分子构造一致（皆为长期负债合计），但股权侧操作化不同。
- 样本：杨芳 (2024) 全 A 股 2011—2022 年（17182 obs）；简冠群 (2025) 专精特新企业子样本。
- 机制路径：杨芳 (2024) = 内部控制 + 信息不对称；简冠群 (2025) 路径需对照确认。

**与 [[李季鹏-2025-耐心资本-高质量发展]] 的差异（共享信息不对称机制）**：

- 因变量不同：杨芳 = NQPF 熵值合成指数；李季鹏 = TFP_LP（高质量发展用 LP 法 TFP）。
- PC 测度：杨芳 = A1（周绍妮换手率三分组 + 长债占比）；李季鹏 = B2（持股稳定性中位数法 + 关系型债权）。
- 信息不对称代理：两者均采用 PCA 流动性指数（流动性比率 + 非流动性比率 + 收益率反转），口径一致，可作为该机制变量的共同操作化。
- 异质性维度差异：杨芳 = 规模 + 股权集中度 + HHI（调节交互形式）；李季鹏 = 规模 + 其他（样本分组形式）。

**与同作者 [[杨芳-2025-耐心资本-制造业-新质生产力-营商环境]] 的差异**：

- 样本范围：杨芳 (2024) 全 A 股；杨芳 (2025) 制造业子样本。
- 调节变量：杨芳 (2024) = 企业规模 / 股权集中度 / HHI；杨芳 (2025) = 营商环境（marketization）。
- 机制：杨芳 (2024) = 内部控制 + 信息不对称；杨芳 (2025) 机制视该论文卡片确认。
- PC 测度同为 A1 经典版（换手率 3 分组 + 长债占比），可视为同一测度框架的"全样本 vs 制造业"两版应用。
