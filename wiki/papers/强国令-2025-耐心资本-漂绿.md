## 研究问题

耐心资本能否抑制企业漂绿（greenwashing）行为？其作用机制是什么？外部环境（环境规制强度、信息化水平）与企业内部特征（所有制、行业污染属性、行业竞争度、股权集中度）如何调节这一治理效应？

## 理论机制

作者从内外部双视角切入资本期限属性对漂绿的约束机制：

- 内部视角：耐心资本的长期持有特性把企业战略锚定在长远可持续价值，使仅服务短期形象的漂绿与长期战略相悖；积极价值倡导通过影响高层决策与组织文化，从内部抑制机会主义漂绿。
- 外部视角：耐心资本的"背书"效应缓解信息不对称，提升真实绿色行为的声誉溢价、加重漂绿被揭露时的声誉惩罚；持续关注环境信息披露质量与减排实效推动信息中介更注重实质评估。

由此推出"内部实质改进—外部积极反馈"的良性循环，落地为三条可检验的中介通道：信息透明度提升、代理冲突缓解、绿色投资促进。

## 研究假设

- 假说 1：耐心资本能够抑制企业漂绿行为（核心，与已有 [[pc-promotes-exploratory-innovation]] / [[pc-shape-exploitative-innovation]] 等耐心资本假设并列但变量不同）。
- 假说 2：耐心资本能够提升信息透明度，进而抑制企业漂绿行为。
- 假说 3：耐心资本能够缓解代理冲突，进而抑制企业漂绿行为。
- 假说 4：耐心资本能够促进绿色投资，进而抑制企业漂绿行为。

调节命题（未编号假说，由表 4 检验）：环境规制强度与信息化水平对耐心资本抑制漂绿的效应有正向调节作用。

## 数据与样本

- 样本区间：2009—2023 年。
- 样本范围：沪深 A 股上市公司。
- 样本筛选规则：剔除金融类、ST/*ST 企业；剔除上市时间 ≤ 2 年的企业；剔除关键变量缺失样本；连续变量上下 1% 与 99% 缩尾。
- 最终样本：13129 firm-year 观测（核心机制检验子样本视变量可用性略有差异：DISCLOSURE 12800、AC 13047、EPI 13120）。
- 数据来源：[[csmar]]（财务、治理、机构持股）、[[wind]]（补充财务与机构数据）、彭博 ESG 评分与华证 ESG 评分（用于漂绿测算，参见 [[greenwashing]]）、《中国环境统计年鉴》《中国固定资产投资统计年鉴》《中国投资领域统计年鉴》《中国城市统计年鉴》《中国能源统计年鉴》（提供调节变量的城市层面数据与上市公司年报）。

## 变量设定

- 核心解释变量：[[patient-capital]] 采用 A4 框架（换手率 2 分组 + 总长债）：
  - 关系型债权 Debt = 长期负债 / 总负债（吴旻佳等 2022 口径，"总长债"对应 A4 框架）。
  - 稳定型股权 Invest = 稳定型机构投资者持股比例；其中机构投资者按每半年度平均换手率分组，高于全样本换手率均值的为交易型，低于均值的为稳定型（周绍妮等 2017，对应 A4 的换手率 2 分组）。
  - 主回归中 Debt 与 Invest 分别替换进入模型（不合并为单一指数），对每条假设各跑一对方程。
- 被解释变量：[[greenwashing]] (GWS)，基于 Yu 等 (2020) 与 Zhang (2022) 思路，由彭博 ESG 评分（"言"）减去华证 ESG 评分（"行"）的行业去均值标准化得分构造（公式 1）：

  GWS_{i,t} = (ESG_{dis,i,t} − \overline{ESG}_{dis}) / σ_{dis} − (ESG_{pre,i,t} − \overline{ESG}_{pre}) / σ_{pre}

  其中 ESG_{dis} 为彭博 ESG 评分（披露端），ESG_{pre} 为华证 ESG 评分（实绩端），均按行业取均值与标准差。
- 中介变量：
  - 信息透明度 DISCLOSURE：深交所对上市公司年度信息披露考评等级 A/B/C/D 赋值 4/3/2/1。
  - 代理冲突 AC：管理费用 / 营业收入（王馨、王营 2021 口径）。
  - 绿色投资 EPI：企业绿色投资 / 总资产（赵领娣、王小飞 2022 口径）。
- 调节变量：
  - 环境规制强度 EG：城市层面政府环境关注度与公众环境关注度的均值。
  - 信息化水平 INF：城市邮电业务总量 / 地区生产总值。
- 控制变量：age（上市年龄）、ROA（净利润 / 平均总资产）、growth（资产增长率）、TA（资产周转率，营收 / 总资产）、Cfo（经营活动现金流 / 流动负债）、TQ（托宾 Q 值，总市值 / 总资产账面值）、CR（大股东持股数 / 总发行股份）。
- 固定效应：行业—年份 + 省份—年份 高维双向固定效应。

## 模型设定

主回归（公式 2）—— 采用 [[two-way-fixed-effects-industry-year]] 的高维固定效应扩展（行业-年份 × 省份-年份）：

GWS_{i,t} = β₀ + β₁ · Debt_{i,t} 或 Invest_{i,t} + γ · Controls_{i,t} + γ_d × ω_t + δ_p × ω_t + ε_{i,t}

机制回归（公式 3）：把被解释变量替换为 DISCLOSURE / AC / EPI，分别检验三条中介通道的第一阶段（PC → M）。

调节回归（公式 4）：

GWS_{i,t} = α₀ + α₁ · (Debt 或 Invest)_{i,t} × Z_{i,c,t} + α₂ · (Debt 或 Invest)_{i,t} + α₃ · Z_{i,c,t} + γ · Controls_{i,t} + γ_d × ω_t + δ_p × ω_t + ε_{i,t}

其中 Z 分别代入 EG（环境规制强度）与 INF（信息化水平）。

## 主要结果

- 基准回归（表 2）：在加入控制变量与"行业—年份"+"省份—年份"双高维固定效应的最严设定下，Debt 系数 −0.412***（t = −3.206，列 3），Invest 系数 −0.001***（t = −2.973，列 6 给出的最简化设定列；带控制变量与高维 FE 的稳定型股权系数同样在 1% 显著为负）。无论是否加入控制变量或固定效应，PC 两个测度的系数均在 1% 水平上显著为负，假说 1 得证。Debt 估计系数绝对值大于 Invest，说明关系型债权的"硬约束"对漂绿的抑制更直接。
- 机制检验（表 3）：Debt 与 Invest 对 DISCLOSURE 系数 0.207*** 与 0.001***（提升信息透明度，假说 2 得证）；对 AC 系数 −0.129*** 与 −0.000***（缓解代理冲突，假说 3 得证）；对 EPI 系数 0.127*** 与 0.000**（促进绿色投资，假说 4 得证）。
- 调节效应（表 4）：Debt × EG 系数 −29.454***、Invest × EG 系数 −0.115**；Debt × INF 系数 −0.529**、Invest × INF 系数 −0.021***。两类调节均在 5% 以上显著为负，验证环境规制强度与信息化水平正向放大耐心资本抑制漂绿的效应。
- 异质性（表 5、表 6）：
  - 所有制：抑制效应在非国有企业组显著，国有企业组不显著（Debt 列）。
  - 行业污染属性：非重污染企业组显著，重污染企业组不显著或弱显著。
  - 行业竞争度：行业竞争度高组系数显著性高于行业竞争度低组。
  - 股权集中度：股权集中度高组系数显著为负，股权集中度低组不显著。

## 机制检验

[[information-transparency-channel]]：耐心资本通过三条通道抑制漂绿：(1) 提升 DISCLOSURE（信息透明度），破坏漂绿赖以生存的信息不对称环境；(2) 降低 AC（代理冲突，以管理费用率代理），削弱管理层短视私利动机；(3) 提升 EPI（绿色投资规模），用可验证的实质投入挤压象征性叙事空间。三条通道在 Debt 与 Invest 两个 PC 测度下均得到一致符号支持。

## 异质性检验

[[ownership-pollution-split]]：四维异质性切分均显示耐心资本治理效应集中在更"治得动"的企业子群——非国有企业、非重污染企业、行业竞争度高、股权集中度高。其逻辑分别为：非国有企业治理结构市场化更易接受耐心资本；非重污染企业有更多低成本环境改善路径；高竞争行业放大漂绿被识破的代价；高股权集中度提供更直接的治理平台把长期价值关切转为对管理层的有效约束。

## 稳健性检验

作者列出三项（限于篇幅未在正文报告完整表，结果留存备索）：

- 替换被解释变量：分别采用选择性披露占比与象征性披露占比的几何平均、漂绿同构指数、企业社会责任报告文本相似性三种替代漂绿度量，Debt 与 Invest 系数仍显著为负。
- 工具变量检验：详见下节"内生性处理"。
- 滞后因变量：将 GWS 滞后一期与两期分别回归，PC 抑制漂绿的效应仍显著，验证长期持续性。

## 内生性处理

[[iv-pension-financial-development]]：选择两组工具变量进行 2SLS——

- 稳定型股权（Invest）的工具变量：2014 年养老金政策的外生冲击。
- 关系型债权（Debt）的工具变量：城市金融发展程度。

两阶段二阶段系数仍在 1% 水平上显著为负；一阶段 F 检验值分别为 42.99（养老金政策 IV）与 16.08（金融发展 IV），通过弱工具变量识别检验。此外作者通过双向高维固定效应（行业—年份 + 省份—年份）控制不可观测的时变共同冲击。

## 可复现线索

- 解释变量构造：
  - Debt = 长期负债 / 总负债，CSMAR 资产负债表"长期负债"与"负债合计"两字段相除即可（A4 框架不细分关系型与非银金融机构借款）。
  - Invest 需先在机构投资者持股层（CSMAR 机构投资者持股表）按机构 ID 计算每半年度换手率，再以全样本换手率均值切分稳定型 / 交易型，最后按公司汇总稳定型机构持股比例。换手率 2 分组的关键阈值是均值而非中位数，与 A1 的 30%/30% 三分组不同。
- 被解释变量构造：
  - 需同时获取彭博 ESG 评分与华证 ESG 评分，两者覆盖期与口径不同，需先按行业（中证或证监会一级行业）计算均值与标准差，再做标准化差分。
  - Stata：可先用 `egen pre_mean = mean(ESG_pre), by(industry year)` 与 `egen pre_sd = sd(ESG_pre), by(industry year)`，再 `gen GWS = (ESG_dis - dis_mean)/dis_sd - (ESG_pre - pre_mean)/pre_sd`。
- 中介变量：DISCLOSURE 来自深交所考评；AC = `MgmtExp / Revenue`；EPI 需手工归集年报附注中的环保设备购置、节能改造支出等口径，公开数据较粗。
- 模型估计：`reghdfe GWS Debt controls, absorb(industry#year province#year) cluster(firm)`；IV 使用 `ivreghdfe GWS controls (Debt = financial_dev), absorb(industry#year province#year)`。
- 注：表 2 列(4)、列(5) 为以 Invest 替换 Debt 的对应模型；列(2) 与列(5) 的 Invest 系数符号在不同列间存在轻微反转，原因是控制变量与固定效应进入顺序不同；推荐以列(3)、列(6) 的最严设定（含控制变量 + 双高维 FE）为主结果。

## 对我当前选题的启发

- 漂绿的"言行差距"测度（彭博 ESG 减华证 ESG，行业内标准化）是本研究项目可直接迁移的被解释变量构造范式；本项目若涉及"双碳"或绿色金融场景，可优先采用此口径。
- 本文 PC 采用 A4 框架（换手率 2 分组 + 总长债），与 [[代飞-2025-耐心资本-双元创新-管理者短视]] 的 A2 框架形成对照，可直接做不同 PC 测算下的稳健性互证。
- 三通道机制（信息透明度、代理冲突、绿色投资）覆盖了治理类研究中最常见的三条传导路径，可作为本项目机制章节的标准模板。
- IV 选择"养老金政策外生冲击 + 城市金融发展"为针对股权 / 债权侧分别设计工具变量提供了清晰范式，避免单一 IV 同时识别两类内生变量的难题。
- 双高维固定效应（行业-年份 + 省份-年份）相对单一两向 FE 更能吸收行业政策冲击与地方经济周期，对 ESG 类研究尤其重要。
