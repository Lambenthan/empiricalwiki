## 研究问题

耐心资本能否通过其长期资金属性缓解中国上市公司的产能过剩问题，进而提升产能利用率？这种作用经由哪些机制（政策扭曲修复、绿色转型驱动、资本配置优化）传导？产业政策与所有制、技术特征、行业产能状况的差异是否带来异质性影响乃至"固化效应"？

## 理论机制

作者基于资本时间属性视角，提出耐心资本通过三条平行机制影响产能利用率：

- 政策扭曲修复机制：耐心资本作为市场化长期资金来源，替代政府短期刺激与补贴，降低企业对政策红利的依赖，使资源配置回归市场信号，缓解"刺激—过剩—淘汰—再刺激"动态失衡。
- 绿色转型驱动机制：耐心资本缓解短期财务压力，使企业有足够资源时间投入绿色技术研发与基础设施改造，提升绿色全要素生产率（GTFP），通过资源效率改善间接化解过剩。
- 资本配置优化机制：耐心资本提高投资门槛与决策审核标准，抑制过度投资和资源错配；但若退化为"懒惰资本"，则可能引发"固化效应"，反向加剧低效产能。

进一步引入产业政策调节：当耐心资本与产业政策（地方一般 / 地方重点 / 国家重点）方向一致时，耐心资本通过"免疫效应"对冲政策执行中的资源错配，强化对产能利用率的提升。

## 研究假设

- H1a：耐心资本有助于缓解企业供需失衡，缓解过剩产能问题。
- H1b：耐心资本会固化低效企业产能，加剧产能过剩问题（竞争性假设）。
- H2：耐心资本会减少市场机制的政策扭曲效应，进而缓解产能过剩。 → [[policy-distortion-mediation]]
- H3：耐心资本会促进企业绿色转型，进而缓解产能过剩。 → [[green-transformation-mediation]]
- H4a：耐心资本通过减少过度投资缓解产能过剩。 → [[overinvestment-mediation]]
- H4b：耐心资本通过加剧过度投资进而加剧产能过剩（竞争性假设）。
- H5：在产业政策的引导下，耐心资本对企业产能利用率的提升作用得以增强。

## 数据与样本

- 样本区间：2008—2023 年。
- 样本范围：中国沪深 A 股上市公司。
- 样本筛选规则：（1）剔除金融企业、上市时间 ≤ 2 年的公司以及 ST 企业；（2）对主要连续变量进行 1% 缩尾；（3）仅采纳审计意见为无保留意见的样本；（4）剔除关键变量存在缺失的观测值。
- 最终样本：42993 个企业-年观测（部分检验因滞后或子样本观测数下降，如 GMM 33260、过剩行业 3541）。
- 数据来源：[[csmar]]（财务、机构投资者持股）、Wind 数据库（上市公司年报与控制变量）、中国研究数据服务平台 CNRDS 产业政策数据库（产业政策态度），辅以《中国环境统计年鉴》《中国固定资产投资统计年鉴》《中国投资领域统计年鉴》《中国城市统计年鉴》《中国能源统计年鉴》等宏观年鉴。本项目的 [[csmar]] 卡片已记录 PC_Sd 持股时长字段；Wind 与 CNRDS 暂不单独建卡。

## 变量设定

- 核心解释变量（两个维度并列）：[[patient-capital]]
  - 战略型股权 PC_Sd（B1 持股时长法）：对每一机构投资者 s 计算样本期内对每家公司 i 的持有总时长 T_si；再求该机构平均时长 T_s = (1/n_s)Σ T_si。若 T_si ≥ T_s，则机构 s 视为公司 i 的战略型投资者；PC_Sd_i 为该公司战略型机构投资者持股比例之和。
  - 关系型债权 PC_Rl：长期负债 ÷ 负债总额（吴旻佳等思路）。
- 被解释变量：[[capacity-utilization]] (CU)，借鉴李雪松等 (2017) 用 SFA Cobb-Douglas 生产函数构建上市公司层面前沿生产面，定义 CU = 实际产出 / 前沿产出 = 技术效率 TE = exp(-μ_it)。投入：资本 K = 总资产，劳动 L = 企业人数，产出 Y = 主营业务收入。稳健性替代变量为总资产周转率 CU_adj = 营业收入 / 总资产。
- 中介变量：
  - 政策扭曲 ID = (一般预算支出 / GDP) × (政府补助 / 总资产)，宏观政府干预度 × 微观补贴强度的交叉项，再做分位数归一化。
  - 绿色全要素生产率 GTFP，借鉴崔兴华和林明裕 (2019)，用 ML 指数 + 非径向 SBM，含期望产出（营业收入）与非期望产出（折算后的工业三废）。
  - 过度投资 OverInv，Richardson 模型正向残差的绝对值（潘越等思路），仅保留正残差样本。
- 调节变量：产业政策 Pol，分地方一般、地方重点、国家重点三个维度。CNRDS 产业政策数据库中政策态度为"鼓励"赋 1，"中性 / 抑制"赋 0；时间维度覆盖"十二五""十三五""十四五"。
- 控制变量：公司年龄、现金流量情况、营业收入增长率、公司规模、托宾 Q 值、资产回报率、产权比率、账面市值比、有形资产比、董事会规模、两权分离度。
- 固定效应：地区（省份）× 年份 + 行业 × 年份 高维双向固定效应。

## 模型设定

主回归（省份×年份 + 行业×年份 高维双向固定效应；不写成独立 model 卡片，沿用本项目 [[two-way-fixed-effects-industry-year]] 的同类思路并在 variant_notes 中记录差异）：

- 模型 (4) 基准：CU_{t,c} = β₀ + β₁·PC_{t,c} + γ·Controls + γ_p·ω_t + δ_i·ω_t + ε_{i,t,c}（PC 分别替换为 PC_Sd 与 PC_Rl）
- 模型 (5)–(6) 政策扭曲中介：ID = β₀ + β₁·PC + γ·Controls + FE + ε；CU = β₀ + β₁·PC + β₂·ID + γ·Controls + FE + ε
- 模型 (7)–(8) GTFP 中介：GTFP = β₀ + β₁·PC + γ·Controls + FE + ε；CU = β₀ + β₁·PC + β₂·GTFP + γ·Controls + FE + ε
- 模型 (9)–(10) 过度投资中介：OverInv = β₀ + β₁·PC + γ·Controls + FE + ε；CU = β₀ + β₁·PC + β₂·OverInv + γ·Controls + FE + ε
- 模型 (11) 产业政策调节：CU = φ + φ₁·PC×Pol + φ₂·PC + φ₃·Pol + φ₄·X + γ_p + δ_i + ω_t + ε

内生性处理：动态面板 GMM，引入 L1.CU 作为工具变量，捕捉产能调整的均值回归特征（[[gmm-dynamic-panel]]）。

## 主要结果

- 基准回归（表 3）：PC_Sd 系数 0.0169–0.0206（1% 显著），PC_Rl 系数 0.0388–0.0507（1% 显著），均显示耐心资本显著提升 CU。引入控制变量与高维 FE 后调整 R² 由 0.02 升至 0.366–0.372。H1a 成立。
- 描述统计：CU 均值 0.751，标准差 0.058，最小 0.035 最大 0.920；PC_Sd 均值 0.777，PC_Rl 均值 0.131；ID 均值 0.509、GTFP 均值 0.993、OverInv 均值 0.055。
- 稳健性（表 4）：替换 CU 为总资产周转率 CU_adj，PC_Sd、PC_Rl 仍 1% 显著为正；替换控制变量（GDP、产业结构、樊纲市场化指数、上市年龄等）后系数仍稳健；滞后 1 期、2 期 PC 仍 1% 显著为正，滞后 1 期效应最大（PC_Sd = 0.0534，PC_Rl = 0.0453）。
- 内生性 GMM（表 5）：L1.CU 系数显著为负（-0.149、-0.209），均值回归特征明显；PC_Sd 系数 0.0469（1%），PC_Rl 系数 0.0105（1%），符号一致。
- 政策扭曲机制（表 6）：PC_Sd 显著抑制 ID（-0.0191，5%），PC_Rl 抑制更强（-0.1501，1%）；ID 对 CU 显著为负（-0.0099 / -0.0087）。Sobel 统计量分别为 2.15 与 8.53，H2 成立。
- 绿色转型机制（表 7）：PC_Sd、PC_Rl 均 1% 显著促进 GTFP（0.0121、0.0311）；GTFP 对 CU 1% 显著为正（0.0203、0.0234）；Sobel = 3.52、6.86，H3 成立。
- 资本配置机制（表 8）：PC_Sd 5% 显著抑制 OverInv（-0.0155），PC_Rl 不显著（0.0148）；OverInv 对 CU 1% 显著为负；Sobel = 2.06（PC_Sd 中介成立）与 -0.81（PC_Rl 中介不显著）。仅 PC_Sd 支持 H4a，PC_Rl 路径未被显著激活。
- 产业政策调节（表 9）：Pol 主效应对 CU 显著为负（地方一般 -0.0043、地方重点 -0.0048、国家 -0.0067，均 1%）；PC×Pol 交互项均显著为正（PC_Sd 框架 0.0414/0.0441/0.0418；PC_Rl 框架 0.0086/0.0125/0.0106），表明耐心资本对冲了产业政策资源错配，"免疫效应"成立，H5 通过。
- 异质性（表 10）：国企组 PC_Sd 不显著（0.0009），非国企 1% 显著（0.0116）；高科技企业 PC_Sd 1% 显著（0.0357），其他企业仅 0.0111；产能过剩行业 PC_Sd 不显著且符号为负（-0.0083），PC_Rl 1% 显著为负（-0.0328）→ 在过剩行业耐心资本表现出"固化效应"，H1b 在过剩行业子样本得证。

## 机制检验

三条并列中介路径：

- [[policy-distortion-mediation]]：PC → ID 抑制 → CU 提升。两类耐心资本均显著，PC_Rl 效应更强。
- [[green-transformation-mediation]]：PC → GTFP 提升 → CU 提升。两类耐心资本均显著。
- 过度投资中介路径（暂不另立 mechanism 卡片）：PC → OverInv 抑制 → CU 提升。仅 PC_Sd 路径显著（系数 -0.0155，5%；Sobel 2.06），PC_Rl 路径未通过 Sobel（-0.81）。

均采用逐步回归 + Sobel 检验。

## 异质性检验

所有制差异 / 技术属性差异 / 产能现状差异（仅产能过剩行业的"固化效应"独立成卡 → [[heterogeneity-overcapacity-industry]]，前两者暂不独立建卡）：

- 所有制：耐心资本仅在非国有企业中显著提升 CU，国企组 PC_Sd 不显著，PC_Rl 仍显著但系数更低。
- 技术属性：参考国家重点支持的高新技术领域目录与《上市公司行业分类指引 2012》对应，将样本切分为高科技企业与其他企业；耐心资本在高科技子样本中作用更强，PC_Sd 系数从 0.0111 跳升至 0.0357。
- 产能现状：参考工信部 2009–2017 淘汰落后产能企业名单匹配 CSRC 行业代码（C22、C25、C26、C28、C30、C31、C32），筛出产能过剩行业 3541 个观测；PC_Sd 不显著且符号为负，PC_Rl 1% 显著为负，证实"固化效应"。

## 稳健性检验

- 替换被解释变量：用总资产周转率 CU_adj = 营业收入 / 总资产 替代 SFA-CU（表 4 列 1–2）。
- 替换/增加控制变量：加入省份层面（GDP、产业结构、樊纲市场化指数）、调整公司层控制（重新计算托宾 Q、上市年龄、营收对数）（表 4 列 3–4）。
- 滞后解释变量：滞后 1 期与 2 期 PC，捕捉延迟效应（表 4 列 5–8）；滞后 1 期效应最显著。
- 全部 PC_Sd、PC_Rl 系数符号与显著性保持一致。本项目暂不单独为这三类稳健性建卡。

## 内生性处理

主识别策略 = 动态面板 GMM（[[gmm-dynamic-panel]]），以 L1.CU 作为工具变量处理被解释变量自相关与潜在双向因果。该策略与代飞 (2025) 的"行业 PC 均值 + 滞后一期" 2SLS 不同；本文未使用 2SLS。注意：作者将"滞后变量"作为稳健性、把"L1.CU GMM"作为内生性独立小节呈现。

## 可复现线索

- PC_Sd 构造：CSMAR 机构投资者持股表按 (机构 s, 公司 i, 季度) 拼接持有起止日期，计算 T_si = 累积持有天数；同时按机构 s 计算其投资过的所有公司平均时长 T_s。Stata 思路：`bysort instid stkcd: egen tenure = total(holding_days)`，再 `bysort instid: egen mean_tenure = mean(tenure)`；标记 `gen strategic = (tenure >= mean_tenure)`；最后 `bysort stkcd year: egen PC_Sd = total(strategic * shareratio)`。
- PC_Rl 构造：CSMAR 资产负债表中 `lt_liab / total_liab`。注意此处与代飞 2025 关系型债务（仅银行长贷分子 + 应付债券应付票据分母）口径不同，需在变量字典中保留两个并列定义。
- CU（SFA）：李雪松等 (2017) 法 — `xtfrontier ln_y ln_k ln_l, ti` 估计 Cobb-Douglas SFA；预测 `predict te, te` 即得 CU。
- CU_adj：营业收入 / 总资产，直接计算。
- ID：宏观财政支出比例需匹配地级市或省份；微观政府补助 / 总资产取自 CSMAR 利润表附注。两者交叉后做分位数归一化（如 `xtile id_q = id_raw, n(100)` → `gen id = id_q/100`）。
- GTFP：MaxDEA 或 R 的 Benchmarking 包做 ML/SBM；非期望产出三废需从《中国环境统计年鉴》取地级市数据再按企业营收占比折算。
- OverInv：先估 Richardson 投资模型，残差 > 0 留作 OverInv 样本。
- Pol（CNRDS 产业政策）：按行业代码（CSRC 2012）匹配五年规划期间政策态度，鼓励 = 1，中性/抑制 = 0。
- 高维 FE：`reghdfe CU PC_Sd controls, absorb(prov#year ind#year)`。

## 对我当前选题的启发

- 本文 PC 框架属于 B1（持股时长法）+ 简化关系型债权，与代飞 2025 的 A2（标准差变体）路径正交。本项目"耐心资本"主分析若按 A2 测算，可把 B1 + 简化债权方案纳入稳健性以测度结果对操作化口径的敏感性。
- 产能利用率的 SFA 测算需要资本、劳动、产出三组数据；CSMAR 的"职工总数"字段缺失率较高，需考虑用"应付职工薪酬"作为劳动投入代理；本项目若沿用 SFA-CU 须在变量字典记录这一数据可得性约束。
- 三条并列中介（政策扭曲 / GTFP / 过度投资）覆盖了"政策—资源效率—投资行为"的完整传导，可作为本项目机制检验的候选模板，尤其 GTFP 和 OverInv 数据获取门槛较低。
- 高维双向固定效应（省份×年份 + 行业×年份）解决了"行业内年份冲击"的同质性问题，比代飞 2025 的"行业+年份"加法形式更严格，但样本要求更高；本项目主回归建议向其靠拢。
- "免疫效应 / 固化效应"二元框架可用于本项目对耐心资本异质性效应的叙事；产能过剩行业的反向显著结果是重要警示，提示在选样时需检查产能过剩行业占比。
- GMM 与 2SLS 是两种独立的内生性处理路径；本项目主分析可保留 2SLS（行业均值 + 滞后），并把 GMM 作为对动态调整的补充识别。
