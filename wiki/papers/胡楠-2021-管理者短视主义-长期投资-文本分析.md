## 研究问题

管理者作为公司战略掌舵人，其内在的"短视主义"（短时间导向）特质是否会导致企业减少资本支出与研发支出等长期投资？这一关系是否会受到公司治理水平、监督型机构投资者持股比例、分析师关注度等内外部监督因素的调节？管理者短视主义是否通过影响长期投资进而影响企业未来绩效？

最核心的方法论问题：如何在中国语境下用可观测的语言文本捕获管理者这种"内在的、稳定的、对时间认知"的特质，规避问卷调研、财务代理变量等已有方法的主观偏差与测量噪声？

## 理论机制

理论基础来自社会心理学的时间导向理论（Lewin 1942；Zimbardo and Boyd 1999）与高阶梯队理论（Hambrick and Mason 1984）：

- 时间导向是先天稳定的个人特质和潜意识过程，决定个体对过去、现在、未来的关注配比；短期导向 → 管理者短视主义。
- 高阶梯队理论：管理者特质决定其行为与战略选择，进而决定企业行为与结果。
- 社会心理学（Webb et al. 1966；Pennebaker et al. 2003）：语言能反映人的认知、偏好与个性，词类与词频可量化人的特质。

由此推出 → 管理者短视主义会使决策视域变短，倾向选择期限短、收益高的项目（Holmstrom 1999；Stein 1988），减少投入在前、收益跨期、风险高的资本支出与研发支出。

## 研究假设

- H1：管理者短视主义与企业长期投资（资本支出与研发支出）呈负相关关系。
- H2a：公司治理水平越高时，管理者短视主义与企业长期投资的负相关关系越易被抑制。
- H2b：监督型机构投资者（证券投资基金、社保基金、QFII）持股比例越高时，该负向关系越易被抑制。
- H2c：分析师关注度越高时，该负向关系越易被抑制。
- 进一步：管理者短视通过减少研发支出和降低资本投资效率，损害企业未来 1/3/5 年绩效（中介效应）。

## 数据与样本

- 样本区间：2007—2018 年（2007 年是中国新会计准则实施起点，避免准则差异）。
- 样本范围：沪深两市全部 A 股上市公司年度财务报告。
- 数据来源：
  - 上市公司年度财务报告 PDF：巨潮资讯网下载。
  - 文本预处理：[[wingo-financial-text-platform]]（WinGo 财经文本数据平台），完成 PDF→TXT、表格剔除、扫描件剔除、MD&A 章节抽取、基于 Jieba + 《英汉现代财会大词典》分词、去停用词、词向量存储。
  - 其他研究数据：[[csmar]]。
- 样本筛选：(1) 剔除金融保险类公司；(2) 剔除 ST/*ST；(3) 剔除上市不超过两年的观测；(4) 剔除缺失值；(5) 连续变量上下 1% 缩尾。
- 最终样本：13,045 个公司-年观测。

## 变量设定

- 核心解释变量：[[managerial-myopia]]（Myopia_Index）— 43 个"短期视域"词词频 ÷ MD&A 总词频 × 100。本论文是该指标的方法学源头（详见"机制检验"与"可复现线索"）。
- 被解释变量：[[long-term-investment]]，分为两个分支：
  - Capex（资本支出 / 总资产）= [购建固定资产、无形资产和其他长期资产支付的现金 + 取得子公司及其他营业单位支付的现金净额 - 处置固定资产/无形资产/其他长期资产收回的现金净额 - 处置子公司及其他营业单位收到的现金净额 - （固定资产折旧 + 无形资产摊销 + 长期待摊费用摊销）] / 年末总资产（借鉴吕长江、张海平 2011）。
  - R&D（研发支出 / 营业收入）= 公司当年研发支出 ÷ 营业收入；研发费用缺失用 0 代替（借鉴张兆国等 2014；虞义华等 2018）。
- 调节 / 分组变量：
  - Gover：公司治理水平。8 个治理指标主成分分析的第一主成分（借鉴张会丽、陆正飞 2012；张学勇、廖理 2010），8 个指标为：第一大股东持股比例、独立董事比例、董事长与总经理是否兼任、监事会会议次数、管理者持股比例、第二到第十大股东持股比例、产权性质、机构持股比例。
  - IO：监督型机构投资者持股比例（仅含证券投资基金、社保基金、QFII）。
  - Analyst：Ln(分析师跟踪人数 + 1)。
- 控制变量：管理者背景（Gage 年龄、Gdegree 学历、Gtenue 任职时间、Gpay 前三高管薪酬总额对数）、企业层面（Size 总资产对数、Cash 现金比率、Growth 营业收入增长率、Leverage 资产负债率、Loss 亏损虚拟变量、ROA、CENT 第一大股东持股、SOE 国有产权虚拟变量、INSD 管理者持股、CEO 二职合一）、外部监督（IO、Analyst）。
- 文本控制变量（稳健性）：Tone 语调、Forward 前瞻性披露水平、Readability 可读性、MD&A_Word 总词频、MD&A_Sent 总句频。

## 模型设定

主回归（OLS + 行业-年度双 FE + 公司层聚类，对应 [[ols-industry-year-fe-cluster]] 的结构变体；本论文未在 wiki 内复用现成 model 页，作者套用李焰等 2011、姜付秀等 2009 的标准设定）：

$$\text{Capex}_{it}/\text{R\&D}_{it} = \alpha_0 + \alpha_1 \text{Myopia\_Index}_{it} + \boldsymbol{\alpha} \cdot \text{Controls}_{it} + \sum \text{Year} + \sum \text{Industry} + \varepsilon_{it}$$

行业按中国证监会 2012 年行业分类标准划分（制造业按门类 + 大类两层细分，其他行业仅按门类）。误差在公司层聚类，使用稳健标准误。

未来绩效模型（中介机制）：

$$\text{ROE}_{i,t+k} = \alpha_0 + \alpha_1 \text{Myopia\_Index}_{it} + \boldsymbol{\alpha} \cdot \text{Controls}_{it} + \sum \text{Year} + \sum \text{Industry} + \varepsilon_{it}, \quad k \in \{1, 3, 5\}$$

中介效应模型采用温忠麟等 (2004) 三步法 + Sobel Z。

资本投资效率（Capex_Residual）由 Richardson (2006) 与吕长江、张海平 (2011) 的预期投资模型残差绝对值得到。

## 主要结果

- 主回归（表 6）：
  - Myopia_Index(t) → Capex：系数 -0.018，t = -2.39（5% 显著）。
  - Myopia_Index(t-1) → Capex：系数 -0.017，t = -2.46（5% 显著）。
  - Myopia_Index(t) → R&D：系数 -0.014，t = -3.27（1% 显著）。
  - Myopia_Index(t-1) → R&D：系数 -0.012，t = -2.77（1% 显著）。
- 描述统计：Capex 均值 0.040，R&D 均值 0.033，Myopia_Index 均值 0.086、中位数 0.069、标准差 0.076；MD&A 平均 3,647 词、180 句。
- 调节作用（表 7）：在公司治理水平高 / 监督型机构持股高 / 分析师关注高三类样本中，Myopia_Index 系数变为不显著；在低组中仍显著为负，H2a-c 全部成立。
- 未来绩效（表 8）：Myopia_Index → ROE1/ROE3/ROE5 系数分别为 -0.031***、-0.025**、-0.021**。
- 中介效应（表 9）：R&D 与 Capex_Residual 均为部分中介；R&D 路径 Sobel Z = -1.777*，资本投资效率路径 Sobel Z = -1.758*。

## 机制检验

中介机制 → [[managerial-myopia-mediation]]：

- 路径一：Myopia_Index → R&D → ROE5（部分中介，Sobel Z = -1.777*）。
- 路径二：Myopia_Index → Capex_Residual（资本投资效率）→ ROE5（部分中介，Sobel Z = -1.758*）。
- 调节作用三类（公司治理、监督型机构持股、分析师关注度）见 H2a-c。

【方法论自我验证】本文为短视文本指标提供四重有效性检验，构成本论文方法学贡献的核心证据：

1. 实际对比法：将"中国上市公司口碑榜"2016–2018 年评选的 14 位"最佳董事长"匹配到样本，对短视文本指标五分位排序后，57%（8 位）落在最低组、86%（12 位）落在 ≤2 组、100% 落在 ≤3 组，与预期相符。
2. 时间维度持续性（Cronbach's Alpha）：12 年未更换董事长企业的 Alpha = 0.861，证明指标具高时序一致性；更换过董事长企业的 Alpha = 0.784，显著低于前者，证明指标捕获的是管理者特质而非企业特征。
3. 跨企业 / 跨董事长比较：同一董事长在两家企业的指标平均差 0.007；同一企业更换董事长前后平均差 0.045。再次确认指标衡量管理者而非企业。
4. 与 Brochet et al. (2015) 的经济后果检验对照（表 2）：短视文本指标与规避亏损正相关、与异常研发费用、异常广告费用负相关、与真实盈余管理正相关，符合 Brochet et al. (2015) 的预期，表明指标有效。
5. 方差分解（Hassan et al. 2019 方法，表 4）：管理者个体固定效应单独可解释短视文本指标 56.4%；环境因素变量仅解释 0.7%；逐步加入年份/行业/公司 FE 后总解释力 58.4%。结论：指标 ≥ 90% 反映管理者内在稳定特质，仅极少部分受环境因素驱动。

## 异质性检验

不属于传统行业 / 产权异质性切分，作者将三类调节因素 (Gover、IO、Analyst) 用样本中位数分成高低组进行分组回归：

- 公司治理水平高低组（[[heterogeneity-corporate-governance-myopia]]）。
- 监督型机构投资者持股高低组（[[heterogeneity-monitoring-institutional-investor]]）。
- 分析师关注度高低组（[[heterogeneity-analyst-coverage-monitoring]]）。

三组结论一致：高监督水平能完全抑制短视的负向作用。

## 稳健性检验

- 滞后一期（同表 6 第 (2)、(4) 列）：[[lagged-myopia-robustness]]，缓解反向因果。
- 内生性两阶段残差介入法（Chen et al. 2013）：[[two-stage-residual-inclusion]]，先估 Myopia_Index 的影响因素回归（含其他短视代理变量 Short_Invest、Turnover、MF），残差作为"增量短视"代入主回归。结果显著负相关（表 10）。
- 控制其他文本机会主义指标：在主回归中加入 Tone（语调，Loughran and McDonald 2011 范式）、Forward（前瞻性披露水平，胡楠、薛付婧 2018）、Readability（基于神经概率语言模型构建的中文文本可读性 = (1/N)·∑ log p_s），结果稳健（表 11），表明短视效应独立于其他文本特征。
- 因变量替换计算方式（表 12）：
  - Capex2 = 固定资产净值改变量 / 年初总资产（李焰等 2011）。
  - Capex3 = (固定资产 + 长期投资 + 无形资产) 净值改变量 / 年初总资产（李万福等 2011）。
  - R&D2 = 研发支出 / 总资产。
  - R&D3 = 研发支出 / 营业收入，但删除研发缺失样本。

## 内生性处理

- 滞后一期短视主义指标：缓解短期内同期相关。
- 两阶段残差介入法：在第一阶段控制其他短视代理变量（Short_Invest、Turnover、MF）与环境因素，第二阶段用残差代入主回归，确保短视文本指标捕获的是管理者特质的"增量信息"（[[two-stage-residual-inclusion]]）。
- 注意：作者未使用工具变量与固定效应（无公司 FE）；这是本论文识别策略的局限，后续应用文献（如代飞 2025）通常会追加公司 FE 与 IV 弥补。

## 可复现线索

【核心—管理者短视主义指标的构建步骤（来自论文第三部分(二)1.）】：

1. 阅读 500 份 MD&A 语料，结合 Brochet et al. (2015) 的英文"短期视域"词集 + Li (2010) 文本指标构建思路，制定中文 MD&A 短期视域种子词集，分两大类：
   - 直接类：天内、数月、年内、尽快、立刻、马上。
   - 间接类：契机、之际、压力、考验。
2. 用 Word2Vec（Mikolov et al. 2013）的 CBOW 模型训练海量中文年度财务报告语料，得到种子词的相似词扩充。CBOW 目标函数为 max ∑_{w∈C} log p(w | Context(w))。
3. 邀请 3 名业界与学术界专家 + 对比 MD&A 文本样例核验，最终确定 43 个"短期视域"词集（详见《管理世界》网络发行版附录 2）。
4. 词典法计算：Myopia_Index = (43 词词频之和 / MD&A 总词频) × 100。值越大表示越短视。
5. 数据预处理依赖 [[wingo-financial-text-platform]]：PDF → TXT、剔除表格（年报中的表格转 TXT 后变文本框，无法直接查询，且表格中数字多、模板化套话多）、剔除扫描文件与缺失文件、抽取 MD&A 章节、基于 Jieba 通用词典 + 《英汉现代财会大词典》等财经词典分词、去停用词、词向量存储。
6. 稳健性词集（论文未公开具体内容，仅在附录中给出）：本指标的扩展或替代版词集，可作为后续研究的稳健性来源。
7. 样本期 2007–2018 年共 13,045 公司-年；MD&A 平均 3,647 词。

【关键变量构造】：

- Capex = [购建固定资产/无形资产/其他长期资产支付的现金 + 取得子公司及其他营业单位支付的现金净额 - 处置固定资产/无形资产/其他长期资产收回的现金净额 - 处置子公司及其他营业单位收到的现金净额 - （固定资产折旧 + 无形资产摊销 + 长期待摊费用摊销）] / 年末总资产。

## 对我当前选题的启发

【方法学奠基地位】本论文是本项目所有"管理者短视主义"中介测度的方法学源头。本项目耐心资本研究将管理者短视作为关键中介机制（已被 [[代飞-2025-耐心资本-双元创新-管理者短视]] 在双元创新场景中实证），所有相关下游论文采用的 Myopia_Index 都直接或间接来自胡楠等 (2021) 的 43 词词典。该论文的有效性检验（实际对比、Cronbach Alpha 跨董事长、跨企业比较、方差分解 56.4%）为本项目使用 Myopia_Index 提供了前提性的可信度证据，使得我在做"耐心资本 → 管理者短视 → ?"的中介推断时不必每次重做方法学验证。

【对耐心资本研究的具体启发】：

1. 中介结构对称：本文 H1 = "短视主义 → 减少长期投资"为负向关系；本项目耐心资本研究的 H 假设 = "耐心资本 → 缓解短视 → 提升长期投资 / 探索式创新"，刚好把本论文倒过来，使得"耐心资本 → 长期投资"路径有了完整的微观认知中介。
2. 调节因素的可迁移性：本文识别的三类监督因素（公司治理水平、监督型机构投资者持股、分析师关注度）直接是耐心资本研究中"机构投资者稳定性、长期持有"的近邻概念。监督型机构投资者持股 = (证券投资基金 + 社保基金 + QFII) / 总股数 这一精确口径，可作为本项目"稳定型机构投资者持股"的对照基准。
3. 长期投资的多种测度选择：Capex 与 R&D 的多种分母（总资产 vs 营业收入）、稳健性中的 Capex2/Capex3，提供了本项目被解释变量的稳健性框架。本项目的"长期投资"如果选 R&D + Capex 两路并行，可以直接复用本论文的稳健性方案。
4. 两阶段残差介入法的可迁移性：本论文用此法验证 Myopia_Index 的"增量信息"。在本项目中，可同样用残差介入法验证耐心资本相对于其他长期资本代理（如长期负债比例、稳定股权比例）的增量信息。
5. 文本数据基础设施的依赖：方法复现严重依赖 [[wingo-financial-text-platform]] + Jieba + 财经词典 + Word2Vec。本项目若需重做或扩展短视指标（例如延伸到 2024 年或换语料），需要保留与之兼容的预处理流水线。

【方法论局限提醒】本论文未做公司固定效应、未做工具变量；这两个识别局限被代飞 (2025) 等下游应用文献部分弥补。本项目在引用胡楠 (2021) 短视指标时，应当明确：(a) 短视指标本身的"内在特质"性质来自胡楠 (2021) 的方差分解证据；(b) 但用该指标做回归时，本项目仍需独立做公司 FE + IV 等识别强化，不能完全依赖原文的滞后 + 残差介入法。
