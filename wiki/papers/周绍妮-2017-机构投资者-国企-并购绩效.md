## 研究问题

中国国有上市公司的机构投资者股东能否提升并购绩效？将机构投资者按持股换手率划分为稳定型与交易型两类后，两类机构投资者在国企并购决策中的治理角色是否存在差异？这种差异在受政府干预较强的关联并购与受干预较弱的非关联并购之间是否进一步分化？

## 理论机制

作者把机构投资者股东积极主义文献中的"积极监督假说""消极监督假说""无效监督假说"三条竞争性命题置于中国国企情境下重新审视，认为机构投资者异质性是三种结论分歧的根源。

- 稳定型机构投资者监督机制：稳定型机构投资者持股期长、对公司经营了解深，理论上有意愿通过"用手投票"影响国企并购决策，从而避免负效应并购，提升并购绩效。
- 交易型机构投资者主动投资机制：作者引入中国实务界"主动型 vs 被动型"分类，指出我国稳定型机构投资者多为指数型基金等被动投资载体（指数基金中稳定型占 77.49%），而交易型机构投资者多为混合型基金（混合基金中交易型占 70.16%），后者激励机制与短期业绩挂钩，反而更有动力参与国企监督与决策。
- 政府干预阻断机制：国企关联并购中政府"干预之手"作用更强，关联并购更易成为体制因素下的价值转移与再分配工具，导致机构投资者无论稳定或交易，其治理作用均被政府意志遮蔽。

三条机制叠加形成本文核心判断：在中国国企情境下，**交易型** 机构投资者反而扮演积极监督角色；其作用仅在政府干预较弱的非关联并购中显现。

## 研究假设

- H1：稳定型机构投资者持股比例与国企并购绩效正相关。 → [[stable-institutional-investors-improve-soe-ma-performance]]
- H2：交易型机构投资者持股比例与国企并购绩效正相关。 → [[transactional-institutional-investors-improve-soe-ma-performance]]

H1 来自股东积极主义主流文献的标准推导（持股稳→治理意愿强→并购绩效高）。H2 来自作者对中国实务界"主动型 vs 被动型"基金分类的引申（稳定型多为被动投资，交易型多为主动投资），论文支持 H2、未支持 H1。

## 数据与样本

- 样本区间：2010-01-01 至 2014-12-31。
- 样本范围：A 股国有上市公司并购交易，实际控制人为国务院国资委、地方国资委或其他政府机构；不含国有股东仅为第一大股东但无实际控制人的样本。
- 样本筛选：
  1. 仅保留上市公司为收购方；
  2. 仅保留并购标的为股权（剔除单项资产收购）；
  3. 剔除并购交易未成功 / 终止；
  4. 剔除金融类样本；
  5. 剔除上市年限不足及样本期间 ST/*ST；
  6. 剔除关键研究数据缺失；
  7. 剔除交易金额 < 100 万元或股权转让比例 < 5% 的小并购。
- 最终样本：358 家上市公司，530 起并购事件（2010 年 75、2011 年 80、2012 年 127、2013 年 131、2014 年 117）。
- 数据来源：
  - 并购事件 → [[csmar]] 上市公司并购重组研究数据库；
  - 机构投资者持股与换手率 → [[wind]]、[[tonghuashun]];
  - 财务与治理特征 → [[csmar]]。
- 缩尾：连续变量上下 1% Winsorize。

## 变量设定

- 被解释变量：[[state-soe-ma-performance]] (ΔADJ_ROE) — 并购首次公告日前后一年净资产收益率（ROE）的差额，剔除同期行业均值。样本均值 −0.01，呈现"并购绩效悖论"。
- 核心解释变量：
  - [[stable-institutional-investors-turnover]] (STABLEINS) — 按 A1 框架（换手率三分组法）取换手率最低组机构投资者在该公司的合计持股比例，使用并购首次公告日前最近半年度或年度末数据。样本均值 1.10%。
  - [[transactional-institutional-investors-turnover]] (TRANSINS) — 按 A1 框架取换手率最高组机构投资者在该公司的合计持股比例，时点同上。样本均值 4.23%。
- 控制变量（均取 t-1 期值，缓解反向因果）：
  - 公司治理：第一大股东持股 (SH1)、独立董事比例 (ID)；
  - 公司财务：资产负债率 (LEV)、自由现金流 / 总资产 (CASH)、总资产收益率 (ROA)、公司规模 (SIZE = ln 总资产)；
  - 并购特征：[[related-party-ma]] (RMA, 关联并购哑变量)、支付方式 (PT, 现金支付 = 1)；
  - 年度虚拟变量 (YEAR, 4 个)。
- 不另设行业固定效应（被解释变量已剔除行业 ROE 均值）。

## 模型设定

主模型 → [[ols-cross-section-ma-event]]：

```
ΔADJ_ROE = β₀ + β₁·STABLEINS + β₂·TRANSINS + Σγ·Controls + Σδ·YEAR + ε
```

观测单位为"并购事件"（一公司可对应多次并购）。估计方法 OLS。论文未明示标准误聚类方式，仅报 t 值。

H1 检验：β₁ 显著大于 0；H2 检验：β₂ 显著大于 0。

异质性切分（[[related-party-ma-split]]）：将 530 起样本按 RMA = 0/1 拆分为关联并购组（293 起）与非关联并购组（237 起），分别重新估计同一方程。

## 主要结果

- 描述性（表 3 Panel B）：
  - ΔADJ_ROE 均值 −0.01、中位数 0、std 0.13、最小 −0.41、最大 0.36 → 国企并购整体绩效呈"并购绩效悖论"。
  - STABLEINS 均值 1.10%、中位数 0.41%、最大 7.59%；TRANSINS 均值 4.23%、中位数 1.19%、最大 31.12%。
  - SH1 均值 41.21% → 国企"一股独大"特征明显。
- 主回归（表 4 列 1，全样本 N=530）：
  - STABLEINS 系数 −0.002（t = −0.61），不显著；H1 未获支持。
  - TRANSINS 系数 0.004（t = 4.00），1% 显著；H2 获支持。
  - 控制变量：ROA 显著为负 (−0.541***)，反映并购前 ROE 高的公司事后均值回归；其它控制基本不显著。
  - R² = 0.157。
- 异质性切分（表 4 列 2-3）：
  - 关联并购样本（N=293）：STABLEINS 系数 −0.000（n.s.）、TRANSINS 系数 0.002（t = 1.58，n.s.），两类机构投资者治理效应均失效。
  - 非关联并购样本（N=237）：TRANSINS 系数 0.004***（1%），方向与全样本一致；STABLEINS 系数 −0.006（n.s.）。
- 推论：
  - 我国稳定型机构投资者多为指数型基金等被动投资载体，未起到对国企并购的制衡与监督作用；
  - 交易型机构投资者多为主动投资型（混合型基金为主），积极参与国企并购决策，提升并购绩效；
  - 关联并购中政府干预较强，遮蔽了交易型机构投资者的治理作用。

## 机制检验

本文未做正式中介检验，但通过以下侧面证据支持 [[stable-investor-monitoring-channel]] 与"主动 vs 被动投资"机制：

- 表 2 证券投资基金内部分布：指数型基金 231 只中稳定型 179 只（77.49%）；混合型基金 630 只中交易型 442 只（70.16%）。基金类型与稳定/交易机构投资者类型高度对应，间接验证稳定型 ≈ 被动投资、交易型 ≈ 主动投资的结构判断。
- 表 4 列 2-3 关联并购 / 非关联并购组对比：交易型机构投资者效应仅在政府干预较弱的非关联并购中显现，间接支持"政府干预阻断治理"机制。

## 异质性检验

按并购是否为关联交易切分（[[related-party-ma-split]]）：

- 关联并购（RMA=1, N=293, 占 55.3%）：通常由政府主导或受政府意志干扰，呈现"报表性重组"或利益输送特征，机构投资者治理失效。
- 非关联并购（RMA=0, N=237）：政府干预较弱，交易型机构投资者治理作用得以体现。

未做行业、地区、并购规模等其他维度切分。

## 稳健性检验

- 工具变量 2SLS（[[iv-industry-mean-stable-region-mean-trans]]，正文叙述未列表）：
  - IVINSIND = 同行业其他国有企业稳定型机构投资者持股均值，作为 STABLEINS 的工具变量；理论依据为稳定型机构投资者存在行业投资偏好。
  - IVINSREG = 同省（直辖市、自治区）其他国有企业交易型机构投资者持股均值，作为 TRANSINS 的工具变量；理论依据为交易型机构投资者通过实地调研形成地区投资偏好。
  - 二阶段结果：交易型机构投资者与并购绩效仍显著正相关，结论稳健。
- 替换被解释变量（[[alternative-ma-performance-measures]]）：
  - 用 ΔADJ_ROA（剔除行业的 ROA 变化量，陈仕华等 2014）；
  - 用 ΔADJ_EBIT（剔除行业的息税前利润率变化量，潘红波和余明桂 2014）；
  - 用并购后两年 ADJ_ROE 均值与并购前两年 ADJ_ROE 均值之差（陈仕华 2014）。
  - 结论均未发生实质性改变（论文未列表）。

## 内生性处理

主回归层面通过 (1) 控制变量取 t-1 期值、(2) 被解释变量采用前后一年差额并剔除行业均值两步缓解反向因果。进一步通过 [[iv-industry-mean-stable-region-mean-trans]] 双工具 2SLS 处理潜在内生性，工具相关性来自机构投资者的行业 / 地区偏好，外生性假设是同行业 / 同地区其它国企的机构投资者持股不通过本企业 PC 之外的渠道直接影响本企业并购绩效。

## 可复现线索

- 机构投资者换手率（A1 三分组）：
  - 半年度时间步：用 [[wind]] 与 [[tonghuashun]] 抓取每个机构投资者每半年累计买入金额 CR_buy 与卖出金额 CR_sell，公式见正文。
  - 单期换手率：CR_{k,t} = 2·min(CR_buy, CR_sell) / Σ(S_{k,i,t}·P_{i,t} + S_{k,i,t-1}·P_{i,t-1})。
  - 平均换手率：研究期内每半年度换手率的算术均值（论文窗口 5 年共 10 个半年度）。
  - 三分组：剔除持股期不足 2 期的机构投资者，按平均换手率排序后等距 3 分组，最高组定义为交易型，最低组定义为稳定型；中间组弃用。
  - 公司层面持股比例：分别计算每只样本股票中稳定型 / 交易型机构投资者的合计持股比例 (STABLEINS / TRANSINS)，时点为并购首次公告日前最近半年度或年度末。
- 被解释变量构造：
  - 取并购首次公告日前一年与后一年净资产收益率差值，再减同期行业 ROE 均值；
  - 行业按 CSRC 行业分类。
- 关联并购识别：CSMAR 并购重组研究数据库内含关联交易标识；交叉核对 t 年公告中 "关联交易" 字段。
- Stata 命令：`reg adj_dROE STABLEINS TRANSINS SH1 ID LEV CASH RMA PT SIZE ROA i.year, robust` 即可复现表 4 列 1；分组样本以 `if RMA == 0/1` 切分。
- 工具变量：`ivreg2 adj_dROE (STABLEINS TRANSINS = ivinsind ivinsreg) controls i.year, first`。

## 对我当前选题的启发

- 本文是 A1 框架（换手率 3 分组 + 改进后的双向半年度换手率算法）在中国国企并购情境下的代表性早期文献。在耐心资本测算时，A1 与 A2（持股标准差变体）、B2（持股稳定性中位数法）属于不同操作化路径；本项目在主分析与稳健性中可叠加 A1 / A2 互证。
- 本文核心反直觉发现：在中国情境下，**稳定型机构投资者 = 被动投资 ≠ 耐心资本治理积极者**。这意味着耐心资本概念在中国语境下不能简单等同于"持股长 = 监督强"，须区分"稳定持股"是否伴随"主动投资意愿"。本项目在写 [[patient-capital]] 的 caveats 时应纳入这一边界条件。
- 政府干预对治理机制的阻断效应：关联并购组所有机构投资者治理变量均失效，提示在国有企业 / 政府背景子样本上做耐心资本异质性时，应单列关联交易、政策性资产注入等政府主导事件做反事实。
- 双 IV 设计（同行业 + 同地区均值）：稳定型与交易型分别使用不同维度均值作为工具，避免单一行业均值同时作为两个内生变量的工具导致欠识别；本项目若使用稳定型 / 交易型连续变量做内生性处理，可借鉴这一思路。
- 并购绩效作为治理效应的"压力测试"：与盈余管理、信息披露等治理代理变量相比，并购涉及大额资源配置决策，是治理效应的高强度检验场景。本项目可考虑在主分析之外加入并购事件子样本，作为耐心资本治理效应的极端情境验证。
