## 研究问题

耐心资本能否促进微观企业高质量发展？信息不对称与第一类代理成本是否分别构成耐心资本影响企业高质量发展的内在传导通道？战略型股权与关系型债权两条路径在效应大小上是否存在差异？

## 理论机制

作者基于股东积极主义理论与信息不对称理论，提出耐心资本通过两条平行机制作用于企业高质量发展：

- 信息不对称通道：耐心资本投资者出于长期持股的尽调需求，会进行更深入的企业价值评估并要求更高的信息透明度；关系型债权投资者通过中间业务（结算、担保、咨询）获得运营信息；战略型股权投资者具有外部信息优势，可向管理层传递"高质量"信号，三条子路径共同降低管理层与外部投资者间的信息不对称程度。
- 第一类代理成本通道：耐心资本提供长期稳定资金缓解短期融资依赖；积极参与公司治理监督管理层；通过股东大会行使投票权约束管理层短期主义倾向，进而缓解所有权与控制权分离造成的代理冲突。

两条机制的最终结果都通过提升研发活动开展（信息不对称通道）和提升综合生产效率（代理成本通道）落地为全要素生产率的提升。

## 研究假设

- H1：耐心资本能够促进企业高质量发展。 → [[pc-promotes-high-quality-development]]
- H2：耐心资本能够通过缓解信息不对称促进企业高质量发展。
- H3：耐心资本可以通过降低企业的第一类代理成本促进企业高质量发展。

## 数据与样本

- 样本区间：2010—2022 年。
- 样本范围：A 股上市公司。
- 样本筛选规则：
  - 剔除金融业企业样本；
  - 剔除资不抵债企业样本；
  - 剔除 ST 和 *ST 企业样本；
  - 剔除数据存在缺失的样本；
  - 连续变量在 1% 与 99% 分位点缩尾。
- 最终样本：文中报告 11046 个观察样本，回归实际使用 11039 条。
- 数据来源：[[csmar]]。

## 变量设定

- 被解释变量：[[high-quality-development]] (TFP_LP)，使用 LP（Levinsohn-Petrin）法估计的企业全要素生产率作为高质量发展的衡量指标，参考苏启林和伍静（2024）。
- 核心解释变量（两个独立变量，分别建模，未合成）：
  - 战略型股权 (Equity)：战略型机构投资者持股比例。判定规则：当年机构投资者持股比例 ÷ 过去 3 年机构投资者持股比例标准差 ≥ 行业-年度中位数 → 战略型；否则归类为交易型机构投资者（参考徐灿宇等 2023）。即 [[patient-capital]] 的 B2 持股稳定性中位数法变体。
  - 关系型债权 (Debt)：上市公司长期负债 ÷ 负债总额。
- 中介变量：
  - 信息不对称 (ASY)：借鉴于蔚等（2012），先构建流动性比率 LR、非流动性比率 ILL、收益率反转 GAM 三个股票流动性指标，再做主成分分析，捕捉与非对称相关的信息合成 ASY。数值越大表示信息不对称程度越高。
  - 第一类代理成本 (AC)：管理费用率（参考周建和袁德利 2013、彭正银和罗贯擎 2022、莫冬燕等 2024）。数值越大表示第一类代理成本越高。
- 控制变量：企业规模 Size、现金流比例 Cashflow、营业收入增长率 Growth、销售净利率 NetProfit、固定资产占比 FIXED、董事会规模 Board、独立董事占比 Indep、两职合一 Dual、托宾 Q 值 TobinQ、人均 GDP（取自然对数）lnPGDP、股权集中度 Top1。
- 固定效应：行业（Industry）+ 年度（Year）虚拟变量。

## 模型设定

主回归 → [[two-way-fixed-effects-industry-year]]：

- 模型 (1)：TFP_LP_it = α₀ + α₁·Equity_it + α₂·Controls + λ + T + ε_it
- 模型 (2)：TFP_LP_it = α₀ + α₁·Debt_it + α₂·Controls + λ + T + ε_it

中介效应（江艇 2022 的两步法）：

- 模型 (3)：TFP_LP_it = α₀ + α₁·Ind_it + α₂·Controls + λ + T + ε_it
- 模型 (4)：Med_it = β₀ + β₁·Ind_it + β₂·Controls + λ + T + ε_it

其中 Ind ∈ {Equity, Debt}，Med ∈ {ASY, AC}。

## 主要结果

- 基准回归（表 3）：
  - 列 (1) 不加控制变量：Equity 系数 1.845***（se=0.046），R² = 0.242。
  - 列 (2) 加控制变量：Equity 系数 1.775***（se=0.049），R² = 0.451。
  - 列 (3) 不加控制变量：Debt 系数 0.199***（se=0.063），R² = 0.138。
  - 列 (4) 加控制变量：Debt 系数 0.408***（se=0.069），R² = 0.392。
  - 战略型股权的估计系数显著大于关系型债权，作者据此认定战略型股权在 PC 促进高质量发展中作用更大，验证 H1。

- 描述性统计（表 2，N=11039）：TFP_LP 均值 8.612、标准差 1.078；Equity 均值 0.565、标准差 0.199；Debt 均值 0.160、标准差 0.171。

## 机制检验

[[information-asymmetry-mediation]]（信息不对称中介，表 4）：

- 列 (2)：Equity → ASY 系数 -0.524***（se=0.025），R² = 0.297。
- 列 (4)：Debt → ASY 系数 -0.361***（se=0.026），R² = 0.282。
- 配合表 3 的 Equity / Debt 对 TFP_LP 显著为正、以及已有研究（宋清华等 2024）信息不对称对高质量发展显著为负的结论，作者按江艇（2022）逻辑判定信息不对称发挥部分中介效应，H2 成立。

[[agency-cost-mediation]]（第一类代理成本中介，表 5）：

- 列 (2)：Equity → AC 系数 -0.030***（se=0.004），R² = 0.567。
- 列 (4)：Debt → AC 系数 -0.035***（se=0.006），R² = 0.568。
- 配合已有研究（李芳等 2024）第一类代理成本降低可显著提高 TFP，作者判定第一类代理成本发挥部分中介效应，H3 成立。

注意：作者采用江艇（2022）"两步走"中介逻辑（验证 X→M、引文佐证 M→Y），未做 Sobel / Bootstrap 直接显著性检验。

## 可复现线索

- 因变量：CSMAR 财务表 + 中间投入数据，调用 Stata 社区命令 `prodest Y, free(L) state(K) proxy(M) method(lp)` 估计 LP 法 TFP；稳健性用 `method(op)`。
- 战略型股权：从 CSMAR 机构投资者持股表取每年机构合计持股比例 inst_ratio，按公司滚动取过去 3 年标准差 inst_ratio_sd_3y；当年比值 ratio = inst_ratio / inst_ratio_sd_3y；按行业（证监会两位代码）-年度计算 ratio 中位数 ratio_med_iy；若 ratio ≥ ratio_med_iy 标记为战略型机构投资者，并加总该机构对该公司的持股比例。
- 关系型债权：CSMAR 资产负债表中 长期负债 ÷ 负债总额。
- 信息不对称 ASY：日频股票交易数据计算 LR、ILL、GAM 三个流动性指标，再做主成分分析（PCA），取第一主成分。Stata：`pca lr ill gam, components(1)`，`predict asy`。
- 第一类代理成本 AC：管理费用 ÷ 营业收入。
- Stata 主回归命令模板：`reghdfe TFP_LP Equity controls, absorb(industry year) cluster(firmid)` 与 `reghdfe TFP_LP Debt controls, absorb(industry year) cluster(firmid)`。
- IV 估计：`bysort industry year: egen equity_ind_mean = mean(Equity) if !mi(Equity)`（剔除本企业版本需先做 sum/count 减自身），再 `ivreghdfe TFP_LP (Equity = equity_ind_mean) controls, absorb(industry year)`。

## 对我当前选题的启发

- 本文将 PC 拆成战略型股权（Equity）与关系型债权（Debt）分别建模，而非合并，这与代飞（2025）A2 求和做法不同；做主回归时可以考虑同时报告"分项 vs 求和"两套结果，让审稿人看到稳健性。
- 战略型股权的判定阈值（行业-年度中位数）来自徐灿宇等（2023），与本项目 wiki 中已有的 B 框架（持股稳定性中位数法）完全对应；该口径可与 A1/A2 形成稳健性矩阵。
- 信息不对称用三个流动性指标 PCA 合成，相对单一指标更稳健，可在本项目稳健性中作为信息不对称代理的备选方案。
- 第一类代理成本用管理费用率而非总资产周转率，理由是后者受规模与行业噪声影响；这一理由可直接照搬。
- 江艇（2022）两步法中介在国内中文期刊普及度高，可与 Sobel / Bootstrap 互为稳健，但需注意作者本人未做直接显著性检验，建议补 Bootstrap 1000 次。
- 同行业 PC 均值 IV 是中文实证标配，但本文报告的第一阶段 F = 1207 与 1194 显著高于 [[谢婷婷-2025-耐心资本-动态能力-绿色转型]] 报告的 t² 量级，意味着该 IV 在长期负债/战略型机构投资者维度上的工具弹性较强，可以放心使用。
- 缺失异质性是该文最显眼的研究空白；本项目可补充按产权性质、规模、行业污染密度切分的检验。
- 仅披露完整稳健性表格、不"留存备索"，这是本项目论文写作的合规底线。
