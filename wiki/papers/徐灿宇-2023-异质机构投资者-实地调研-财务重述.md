## 研究问题

同交易型机构投资者相比，稳定型机构投资者是否能够抑制上市公司的财务重述行为？机构投资者的实地调研活动在异质机构投资者治理财务重述行为的过程中起到何种调节作用？信息环境与内部控制是否构成了稳定型机构投资者抑制财务重述的中介路径？

## 理论机制

稳定型机构投资者持股份额较大、期限较长，更加注重公司可持续经营，借助自身资源和信息优势监督管理层；交易型机构投资者持股期限短、投机动机强，可能与管理层合谋掩饰坏消息。两类机构投资者治理意愿与效力存在显著差异：

- 信息环境路径：稳定型机构投资者通过持续接触公司经营信息，降低管理层操纵盈余的能力，从而改善公司内部信息透明度（以可操控应计利润绝对值刻画）。
- 内部控制路径：稳定型机构投资者具有更强的动机推动公司内部治理建设，提升内部控制质量（以迪博内部控制指数刻画）。
- 实地调研强化：实地调研使机构投资者通过现场参观、面对面问答等方式获取私有信息，并形成不定时持续监督，从两方面强化稳定型机构投资者的治理效力。

## 研究假设

- H1：同交易型机构投资者相比，稳定型机构投资者对财务重述行为具有抑制作用。
- H2：实地调研强化了稳定型机构投资者对财务重述行为的抑制作用。

## 数据与样本

样本区间为 2013—2020 年（深交所 2013 年起要求上市公司编制《投资者关系活动记录表》）；样本范围为深圳证券交易所 A 股上市公司；剔除 ST 与 *ST、金融类公司、无机构投资者持股以及数据缺失或异常的公司，最终得到 13558 个公司—年度观测。连续变量进行上下 1% 缩尾。

数据来源：

- 财务重述数据 → 迪博数据库
- 机构投资者持股相关数据 → 锐思 RESSET 数据库
- 公司财务及特征 → 国泰安 CSMAR
- 内部控制指数 → 迪博内部控制指数

## 变量设定

被解释变量：财务重述（Restate）—— 当年发生财务重述行为取 1，否则为 0。

核心解释变量：异质机构投资者（Stable）—— 采用持股稳定性中位数法（B2 框架）。先计算 SD = 机构投资者持股比例 / 过去三年机构投资者持股比例标准差，若公司当年 SD ≥ 行业年度中位数则 Stable=1（稳定型），否则为 0（交易型）。

调节变量：实地调研（Vist）—— 公司当年发生过机构投资者实地调研活动取 1，否则为 0。

中介变量：

- 信息环境（DA）：当年可操控应计利润的绝对值（Dechow et al.）
- 内部控制（IC）：迪博内部控制指数（梁上坤）

控制变量：公司规模（Asset = ln 资产总额）、财务杠杆（Lev）、成长性（Growth）、盈利水平（Roe）、前 5 大股东持股（Sh5）、管理层持股（MShare）、两职兼任（Duality）、董事会规模（Boardsize）、独立董事比例（IndB）、上市年龄（Listyear）、审计意见（Audit）、"四大"审计（Big4）。

固定效应：年度（Year）+ 行业（Industry）。

## 模型设定

H1 检验（基准）：

```
Restate = α0 + α1·Stable + Σγ·Control + ΣIndustry + ΣYear + ε   ... (2)
```

H2 检验（调节）：

```
Restate = β0 + β1·Stable + β2·Stable×Vist + β3·Vist + Σλ·Control + ΣIndustry + ΣYear + ε   ... (3)
```

机制检验（中介）：

```
DA / IC = β0 + β1·Stable + Σλ·Control + ΣIndustry + ΣYear + ε    ... (4)
Restate = γ0 + γ1·Stable + γ2·DA / IC + Σλ·Control + ΣIndustry + ΣYear + ε   ... (5)
```

估计方法：模型 (2) (3) (5) 用 Logit 估计；模型 (4) 中 DA/IC 作为连续变量用 OLS 估计。标准误为公司层面聚类稳健标准误。

## 主要结果

H1（表 3 列 1）：Stable 系数 −0.217，1% 显著。Stable 每增加一个标准差，财务重述概率下降 1.18%（相对均值下降 8.67%）。稳定型机构投资者显著抑制财务重述。

H2（表 3 列 4）：Stable×Vist 系数 −0.199，10% 显著；Stable 主效应在调研组（列 2）系数 −0.295（1% 显著），未调研组（列 3）系数 −0.094（不显著）。实地调研强化了稳定型机构投资者对财务重述的抑制作用。

控制变量：Asset、Roe、Audit 系数显著为负；Lev、Listyear 系数显著为正；Sh5、MShare、Boardsize 不显著。

## 机制检验

表 4 信息环境路径（列 1–3）：Stable→Restate 系数 −0.026（1%）；Stable→DA 系数 −0.002（5%）；同时纳入 Stable 与 DA 时，Stable 系数 −0.026（1%），DA 系数 0.235（1%）。信息环境构成部分中介。

表 4 内部控制路径（列 4–6）：Stable→Restate 系数 −0.025（1%）；Stable→IC 系数 0.014（5%）；同时纳入时 Stable 系数 −0.024（1%），IC 系数 −0.081（1%）。内部控制构成部分中介。

结论：稳定型机构投资者通过提升信息透明度（降低 DA）与改善内部控制水平（提升 IC）抑制财务重述。

## 异质性检验

按财务重述内容分为技术问题型 Restate_t（录入、校对、排版、统计、计算等技术原因）与非技术问题型 Restate_nt（会计问题、敏感问题等）；按经济后果分为欺诈型 Restate_f（同时违规）与非欺诈型 Restate_nf。

表 7：稳定型机构投资者对四类财务重述均有显著抑制作用（Stable 系数：Restate_t −0.210∗∗、Restate_nt −0.184∗∗、Restate_f −0.183∗、Restate_nf −0.213∗∗∗）。

表 8：实地调研仅强化了对非技术问题型与非欺诈型财务重述行为的抑制作用（Stable×Vist 在 Restate_nt 列系数 −0.289∗∗、在 Restate_nf 列系数 −0.225∗）；技术问题型与欺诈型样本中交乘项不显著。

## 稳健性检验

- 公司固定效应模型（缓解不随时间变化的遗漏变量）：Stable −0.206∗∗∗，Stable×Vist −0.242∗∗。
- 改变被解释变量：以财务重述频率（一年内重述公告次数）替换虚拟变量：Stable −0.202∗∗∗，Stable×Vist −0.186∗。
- 改变回归方法：Probit 替代 Logit：Stable −0.120∗∗∗，Stable×Vist −0.105∗。
- 稀有事件 Logistic 模型（应对 Restate 均值仅 0.136）：Stable −0.217∗∗∗，Stable×Vist −0.199∗。
- 补对数—对数模型：Stable −0.198∗∗∗，Stable×Vist −0.186∗。

## 内生性处理

- 倾向得分匹配（PSM）：以 Asset、Lev、Roe 等为协变量做 1∶1 匹配，得到处理组与控制组共 3680 个观测重新回归。Stable −0.215∗∗∗，Stable×Vist −0.272∗。
- Heckman 两阶段：第一阶段以"是否有稳定型机构投资者持股"为被解释变量估计概率；第二阶段在原回归中加入逆米尔斯比率（IMR）。Stable −0.216∗∗∗，Stable×Vist −0.204∗，IMR 在 10% 水平显著。

两种方法处理后基准结果保持稳健。

## 可复现线索

- 数据库：迪博（财务重述、内部控制指数）、RESSET 锐思（机构投资者持股，特别是分类稳定型/交易型所需的持股比例与过去三年标准差）、CSMAR（公司财务、治理结构、审计意见）。
- Stable 测算：先按公司—年度计算机构持股比例 / 过去 3 年持股比例标准差 → SD；按行业—年度求 SD 中位数；与中位数比较生成 0/1 哑变量。
- DA：使用修正 Jones 模型估计可操控应计利润，取绝对值。
- 实地调研：根据深交所《投资者关系活动记录表》披露的机构投资者实地调研事件，按"当年是否发生"生成 0/1 哑变量。
- 估计：Logit + 公司层面聚类稳健标准误；行业与年份固定效应；表 3 报告 Robust Z 值。
- Stata 边际效应：用 prchange 命令计算 Stable 每增一个标准差的概率变化。

## 对我当前选题的启发

- 持股稳定性中位数法（B2）属耐心资本"股权侧 / 时间属性"操作化变体之一：与 A2 的"机构持股 / 持股标准差"原始连续指标互补，本文将其转为行业—年度中位数二分类，避免对原始稳定性数值分布的依赖，更适合作 Logit / PSM。
- 实地调研作为外生信息冲击：可作为耐心资本治理效应的一种检验渠道——耐心资本是否需要"实地交互机会"才能转化为治理效力？
- 异质性切分财务重述类型：技术性 vs 非技术性、欺诈型 vs 非欺诈型的二维划分可以借鉴用于刻画信息治理边界。
- 中介模型采用经典逐步法（DA/IC 分别检验），与本项目的中介稳健性框架兼容。
