## Definition

新质生产力（New Quality Productive Forces, Npro）是以创新为主导的先进生产力，强调技术革命性突破、生产要素创新性配置与产业深度转型升级的有机结合。理论上由三大要素构成：

- 新质劳动者：以高素质创新型人才为代表，包括高学历研发人员、具备数字化能力的管理团队。
- 新质劳动对象：从生态环境与未来发展两个层面拓展生产对象，涵盖生态资源、固定资产结构、智能化生产基础设施。
- 新质劳动资料：以科技、绿色、数字三类劳动资料为载体，体现为创新产出、绿色技术、数字资产。

## Measurement

主流路径为多指标 + 熵权法综合得分。具体子指标随研究主题（科技强国、绿色化、数字化）权重不同。本项目目前记录的口径为简冠群 (2025) 测度。

## Data Source

- [[csmar]]：财务数据、高管背景、研发人员占比、固定资产占比、机构投资者持股。
- [[cnrds]]：华证 ESG 环境得分、专利数据、绿色投资者库、数字化与智能化文本词典。
- [[wind]]：补充财务、市场指标、智能化设备相关数据。
- 企业年报全文：用于词频法计算智能化、数字化、绿色化、供应链相关词频。
- IFR 国际机器人联合会：行业层机器人安装量数据，与 CSMAR 行业岗位结构结合得到企业层机器人渗透率。

### 简冠群 (2025) 测度（本论文）

借鉴张秀娥等 (2025) 构建 14 个底层指标，分四个维度：

- 新质劳动者
  - 员工素质 — 研究生学历占比 = 研究生学历员工 / 总员工。
  - 高素质员工 — 研发人员占比 = 研发人员 / 总员工。
  - 管理层数字化背景 — 高管团队是否具有数字化背景哑变量（有 = 1）。
  - CEO 职能经历丰富度 — CEO 职能经历计数。
- 新质劳动对象
  - 环境绩效 — 华证 ESG 评分体系中的环境得分（[[environmental-performance]]）。
  - 固定资产占比 — 固定资产 / 资产总额。
  - 机器人渗透率 — 行业层机器人渗透度 × 企业生产部门员工占比。
- 新质劳动资料
  - 科技劳动资料：企业创新水平 = Ln(授权专利数 + 1)。
  - 绿色劳动资料：绿色技术水平 = Ln(授权绿色专利数 + 1)；绿色专利占比 = 授权绿色专利 / 授权专利。
  - 数字劳动资料：智能化水平 = Ln(智能化词频 + 1)；数字资产占比 = 数字相关资产 / 无形资产总额。
- 优化组合（数字发展、绿色发展）
  - 数字化创新 = Ln(年报中数字化技术创新 + 数字化管理创新词频 + 1)。
  - 供应链数字化 — 是否参与供应链创新与应用试点（哑变量）。
  - 持续绿色创新水平 — (t-1 与 t 年绿色专利申请量之和较 t-2 与 t-1 年的环比增长率) × (t-1 与 t 年的绿色专利申请量之和)。
  - 绿色投资者数量 = Ln(绿色投资者数量 + 1)。

合成方法：14 个底层指标分别标准化（min-max 或 z-score），熵权法计算权重并合成单一得分，最后乘以 10。简冠群 (2025) 描述性统计中 Npro 均值 1.801、SD 1.102、范围 [0.0238, 4.987]。

## Construction Steps

简冠群 (2025) 测度复现：

1. 从 CSMAR 上市公司高管表抽取员工学历、研发人员、CEO 职能经历、高管数字化背景。
2. 从 CSMAR 财务表取固定资产、资产总额、研发支出。
3. 从 CNRDS / 华证 ESG 数据库取年度环境得分。
4. 用 IFR 行业机器人安装数据 × CSMAR 行业 - 企业生产部门员工占比，得到企业层机器人渗透率。
5. 从 CNRDS 专利库取企业当年授权发明 / 实用新型 / 外观设计专利及绿色专利数量。
6. 从企业年报全文做词频统计：智能化、数字化技术创新 + 管理创新词典；绿色相关词典在持续绿色创新与绿色投资者中体现。
7. 数字资产占比基于无形资产明细中数字化相关项识别。
8. 14 个底层指标分别标准化。
9. 熵权法计算各指标权重 → 加权合成 → 乘以 10 → Npro。

## Stata Notes

```stata
* 假设已合并 14 个底层指标 x1-x14
* 标准化（min-max）
foreach v of varlist x1-x14 {
    egen `v'_min = min(`v')
    egen `v'_max = max(`v')
    gen `v'_n = (`v' - `v'_min) / (`v'_max - `v'_min)
}

* 熵权法权重（用户自定义 ado 或手算）
* 略：参考王明涛 (1999) 多指标综合评价熵值法 ado

gen Npro = 10 * (w1*x1_n + w2*x2_n + ... + w14*x14_n)
```

## Caveats

- 14 个底层指标对数据可得性要求较高（华证 ESG、IFR 机器人数据、年报词频字典），缺失任一指标都需重赋权重，导致跨研究不可直接比较。
- 熵权法对极端值敏感，连续变量须先做 1% 缩尾。
- 词频法依赖词典明细；不同论文使用的关键词字典差异显著（绿色化、智能化、数字化），跨论文复现需公布词典。
- 持续绿色创新水平的环比增长率分母为零或负时数值不稳定，需做技术处理。
- "× 10" 仅是缩放，便于解读，不改变估计系数显著性，但报表呈现需统一。
- 与传统全要素生产率（[[total-factor-productivity]]）相比，Npro 强调创新主导与三大要素融合，单一 TFP 不能完全替代。
- 企业层机器人渗透率受行业层机器人数据频率限制，年度内变动主要来自岗位结构。
