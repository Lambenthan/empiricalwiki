## Purpose

本检验有两层目的：

1. **稳健性验证**：通过子样本回归确认主效应在不同时段下均显著为正，排除"特定时段驱动"的可能。
2. **政策周期信号识别**：2017 年是中国宏观政策与资本市场环境重要分水岭——党的十九大提出新发展理念、供给侧结构性改革深化、新经济周期开启、A 股资本市场改革（IPO 节奏调整、科创板筹建、注册制试点）加速。比较 2011-2016 与 2017-2022 两段中耐心资本系数的差异，可以识别耐心资本在新质生产力培育中的作用是"持续稳定"还是"边际增强"。

## When To Use

- 当核心解释变量（如耐心资本、ESG、数字化转型）在样本期内经历显著政策环境变化时。
- 当研究者关心"机制是否在政策周期前后稳定"或"作用强度是否随制度环境演化而增强"。
- 与 PSM、安慰剂检验等组合形成稳健性最小套件，覆盖不同潜在偏误来源。

## Implementation

杨芳等（2024）表 4 Panel B：

- 样本切分：以 2017 年为分界，2011-2016 子样本（n = 5468/5467）、2017-2022 子样本（n = 11746/9210）。两段子样本量不一致主要因为不同时段的可观测样本量与缺失模式不同，且滞后一期处理使 2011 年作为初始观测年损失部分样本。
- 模型：与基准模型相同（OLS + 行业 FE + 年份 FE + 10 个企业层控制变量 + 1% 缩尾 + 滞后一期处理）。
- 比较：在两段子样本中分别报告稳定型股权与关系型债权的系数与 t 值，比较系数绝对值与显著性水平变化。

```stata
* 第一段子样本（2011-2016）
reghdfe Npro L.Wd $controls if year >= 2011 & year <= 2016, ///
    absorb(industry year) cluster(firmid)
reghdfe Npro L.Pc $controls if year >= 2011 & year <= 2016, ///
    absorb(industry year) cluster(firmid)

* 第二段子样本（2017-2022）
reghdfe Npro L.Wd $controls if year >= 2017 & year <= 2022, ///
    absorb(industry year) cluster(firmid)
reghdfe Npro L.Pc $controls if year >= 2017 & year <= 2022, ///
    absorb(industry year) cluster(firmid)
```

## Expected Table Pattern

| 变量 | 2011-2016 | 2017-2022 |
|------|----------|-----------|
| 稳定型股权 | 5.5993∗∗（t = 1.95） | 26.1733∗∗∗（t = 10.17） |
| 关系型债权 | 0.1382∗（t = 1.95） | 1.1620∗∗∗（t = 4.09） |
| 观测值 | 5468 / 5467 | 11746 / 9210 |
| 调整 R² | 0.239 / 0.233 | 0.391 / 0.390 |

## Interpretation

- 两段子样本中"稳定型股权"和"关系型债权"对"新质生产力"的回归系数均显著为正，再次表明本文分析结果稳健。
- **2017—2022 子样本系数的显著性和绝对值均明显大于 2011—2016 子样本**，表明：
  - 稳定型股权效应：从 5.60（5% 显著）跃升至 26.17（1% 显著），绝对值放大约 4.7 倍。
  - 关系型债权效应：从 0.14（10% 显著）升至 1.16（1% 显著），绝对值放大约 8.4 倍。
- 经济解读：随着时间推移，耐心资本增加对企业新质生产力发展的促进作用趋于增强。这一变化可能源自：
  - 供给侧结构性改革推进，企业转型需求增加，长期资金更具战略价值。
  - 资本市场改革（科创板、注册制、退市制度完善）使长期资金退出渠道改善，耐心资本扩张更迅速。
  - 新质生产力（数字化转型、绿色技术、AI 采纳）相关投入在 2017 年后大量释放，耐心资本恰好对接需求。
  - 政策导向（"双碳"目标 2020、新质生产力概念 2024）逐步落地，耐心资本与政策协同效应增强。

## Caveats

- **样本量不平衡**：2017-2022 子样本观测数明显多于 2011-2016（约 2 倍），子样本系数比较应注意标准误估计的精度差异。
- **行业 / 公司构成变化**：2017 年后科创板上市公司、新经济企业大量进入样本，可能改变企业类型分布；建议同时控制公司层固定效应或在子样本内重新做趋势分析以排除"公司组合效应"。
- **政策事件累积**：2017-2022 段同时叠加多个政策（双碳、注册制、新质生产力等），无法识别具体哪一政策驱动了系数增强；要做精确事件识别需配合 DID 设计。
- **指数体系动态变化**：新质生产力指数所用 27 个三级指标（数字化、AI、绿色技术等）在 2011-2016 段数据可得性较差（如 AI 采纳、数字技术创新数据 2017 年后才大量披露），可能导致前段观测的 NQPF 估计偏低，进而放大后段系数。
- **未做正式 Chow 检验**：论文未给出两段系数差异的统计检验（如 Chow Test、交互项检验），仅做描述性比较；建议本项目复用时补充正式检验。
