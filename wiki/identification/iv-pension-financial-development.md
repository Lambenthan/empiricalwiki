## Identification Problem

耐心资本（Debt 与 Invest）的取值不是外生分配，而是企业 / 城市 / 行业层面众多内生选择的均衡结果——漂绿严重的企业可能因声誉受损被长期投资者回避，反过来又显得"PC 低 + 漂绿高"，造成反向因果与遗漏变量偏误。需要工具变量隔离 PC 对漂绿的因果方向。

## Strategy

针对 PC 的两个测度分别选择工具变量：

- 稳定型股权 Invest 的 IV：2014 年养老金政策外生冲击。养老金属于天然的长期机构投资者，其入市政策对稳定型股权供给构成外生扰动。
- 关系型债权 Debt 的 IV：城市金融发展程度。城市层面的金融基础设施越发达，企业越易获得长期银行信贷与债券融资，但城市金融发展程度不直接进入企业漂绿的微观决策方程。

每个 IV 配合一个内生变量做 2SLS，二阶段系数仍均显著为负，与基准回归方向一致。

## Key Assumptions

- 相关性：一阶段 F 值 42.99（养老金政策 IV）与 16.08（金融发展 IV）均高于 Stock-Yogo 弱工具识别阈值 10，满足相关性。
- 排他性：
  - 养老金政策被假定通过改变长期机构资金供给路径影响 Invest，而非通过其他渠道直接影响漂绿。需要警惕的是 2014 年同期可能存在 ESG 相关政策叠加，作者通过双高维固定效应（行业-年份 + 省份-年份）吸收时变共同冲击。
  - 城市金融发展通过改变企业长期信贷可得性影响 Debt，需要警惕地方金融发展可能伴随地方环境监管强度变化；同样通过省份-年份固定效应缓解。
- 单调性：养老金政策对所有企业的影响方向一致（增加稳定型股权供给）；金融发展对所有企业 Debt 的影响方向一致（增加长期负债占比）。

## Implementation

- 工具变量数据：城市金融发展程度通常可用城市金融机构存贷款余额 / GDP 度量；养老金政策冲击可构造为政策时点后乘以企业初始养老金敞口的 Bartik 风格变量（论文未给出完整构造细节，原文以"2014 年养老金政策外生冲击"直接表述）。
- Stata 命令骨架：

```
* 稳定型股权 IV 回归
ivreghdfe GWS controls (Invest = pension_shock_iv), absorb(industry#year province#year) cluster(firm)

* 关系型债权 IV 回归
ivreghdfe GWS controls (Debt = city_finance_dev), absorb(industry#year province#year) cluster(firm)
```

## Diagnostics

- 一阶段 F 检验：养老金政策 IV 一阶段 F = 42.99（稳定型股权方程）；金融发展 IV 一阶段 F = 16.08（关系型债权方程）。两者均通过弱工具检验。
- 二阶段系数：仍在 1% 水平上显著为负，符号与基准回归一致。
- 论文未报告过度识别检验（每个内生变量仅一个工具变量，无法做 Sargan / Hansen 检验），这是该识别策略的局限。

## Limitations

- 单一工具难以同时检验排他性，需依赖理论论证而非统计检验。
- 养老金政策的"外生性"在 2014 年实施前后的预期效应可能污染前期样本；建议作为稳健性时考虑剔除政策窗口期。
- 城市金融发展同时会通过其他渠道（如绿色信贷、绿色债券）影响企业绿色行为，排他性较弱；建议补充绿色信贷政策作为额外稳健性 IV。
