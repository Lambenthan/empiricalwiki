## Identification Problem

李心武 (2025) 检验耐心资本对企业数字化转型的认同效应（H2）时，存在两类内生性威胁：

- **遗漏变量**：地方经济发展、产业政策、企业治理特征等可能同时影响数字化转型水平与基金持仓行为。
- **双向因果**：耐心资本的持仓本身可能反向激励企业加大数字化转型投入，造成 PC 持仓 ↔ 数字化转型互为因果。

## Strategy

构造两个外生工具变量同时进入两阶段最小二乘法（2SLS）：

- **IV1：光缆铺设长度**。参考王海等 (2023)。光缆是网络布局中重要的通信技术设施，省级光缆铺设长度反映当地通信基础。计算 ln(光缆铺设长度)。
  - 相关性：地区数字基础设施越好，当地企业越易开展数字化转型。
  - 外生性：光缆铺设由历史规划与人口分布决定，不会因为某只基金对某家企业的持仓行为而改变。
- **IV2：政府工作报告数字化关注度**。通过 Python 对省级政府工作报告分词，统计与数字化转型相关词汇数量，再 + 1 取对数得到 IV2。
  - 相关性：地方政府关注度高的区域，企业数字化转型推进更快。
  - 外生性：基金依据政府工作报告对单一企业的持仓决策的直接概率较小。

## Key Assumptions

- IV1 与 IV2 均与企业数字化转型显著相关（第一阶段 F 统计量需通过弱工具变量检验）。
- IV1 与 IV2 均通过外生性条件（与残差不相关）。
- 两个工具变量同时使用可执行 Hansen J 过度识别检验，验证至少有一个工具变量外生。

## Implementation

第一阶段：

`Digital_{j,t} = π_0 + π_1 · IV1_{p,t} + π_2 · IV2_{p,t} + γ · Controls + Fund × Firm + Year + ε`

第二阶段：

`Proportion_{i,j,t} = β_0 + β_1 · Digital_Fit_{j,t} + γ · Controls + Fund × Firm + Year + ε`

李心武 (2025) 检验结果：

- **第一阶段**：两个 IV 与 Digital 均显著相关。
- **第二阶段**：Digital_Fit 系数显著为正，说明剥离反向因果与遗漏变量后，数字化转型仍提高耐心资本持仓比例。
- **诊断检验**：
  - Kleibergen-Paap rk LM 统计量 = 125.262，通过不可识别检验。
  - Kleibergen-Paap rk F 统计量 = 62.955，通过弱工具变量检验。
  - Hansen J 检验 P 值 = 0.1910，不拒绝原假设，通过过度识别。
  - Durbin-Wu-Hausman 统计量 = 4048.97，验证内生性存在。

## Diagnostics

- 第一阶段 R² 与 F 统计量。
- 弱工具变量 F > 10。
- 过度识别 Hansen J P 值 > 0.10。
- DWH 检验确认内生性。

## Limitations

- 光缆铺设长度作为省级 IV，无法识别省内不同企业之间的横截面差异，需结合企业层面控制变量。
- 政府工作报告词频的省级层面也存在样本聚合损失。
- IV 估计反映的是局部平均处理效应（LATE），系数大小不能直接外推到所有企业。
