## Grouping Logic

按实际控制人是否为国有单位划分两组并分别回归，作者论文里的所有 5 张表（INST、INST_LONG/SHORT、BHAR、ΔLOAN/LCOST、ETR）均在国有 vs 非国有子样本上重复完整方程。

- STATE = 1：实际控制人为国有单位（含中央国资委、地方国资委、央企集团及其下属公司）。
- STATE = 0：民营、外资、集体、其他。
- 黎文靖 (2015) 全样本 STATE 均值 0.641，即 64.1% 观测为国有控股。

## Theoretical Rationale

「政企关系密度」是该划分的核心理论依据：

- **国有企业**：政府对其有直接的人事、财政、信贷、税收政策渗透能力。良好的环境绩效会被地方政府视为「配合中央生态文明战略」的体现，政府以银行信贷优惠和所得税减免作为奖励，长期机构投资者能识别此种长期价值。
- **非国有企业**：市场化程度更高，政府支持渠道难以直接传导；企业的环境投入更可能是市场压力（消费者、上游客户、机构投资者的 ESG 偏好）的产物，而非政策博弈结果。论文证据显示，非国有企业的 LCOST 反而随 LNENV 上升，即环境投入抬高融资成本，与「政府不补贴」一致。

## Model

每个子样本独立运行 [[ols-industry-year-fe-cluster]]，控制变量、固定效应、聚类设置完全相同；分组只通过样本筛选（`if STATE == 1`）实现，未引入 LNENV × STATE 交互项。论文未对组间系数差异做正式 χ²、suest 或 bootstrap 检验，组间显著性差异基于直接观察。

## Interpretation

- **政策含义**：政府对国有企业的环境表现有明显「奖励」机制，对非国有企业则缺乏类似激励。后续政策若希望鼓励非国有企业改善环境，需要建立独立于所有制的奖励渠道（如绿色信贷、绿色债券）。
- **研究启示**：单维所有制切分是中国实证最经典的异质性切分之一，但仅能反映「政企关系」一个机制；对于本项目耐心资本研究，需要补充行业污染属性、行业竞争度等更多维度（参见[[ownership-pollution-split]]）。
- **局限**：组间系数差异未做统计检验；STATE 的内生性未处理（实际控制人可能与企业环境投入决策同时受地方政府影响）。
