## Scope

WinGo（文构）是面向中美上市公司披露文本的财经文本数据平台，四大核心库：**中国上市公司**（2001 年起，年报 / 季报 / MD&A / 审计报告 / 附注 / 招股书 / 内控评价 / 业绩说明会 / 社会责任报告 / 风险章节，350 万+ 文档）、**美国上市公司**（1993 年起）、**中国政府文本**（国务院政府工作报告 1954 年起，省地级 2001 年起）、**中国专利**。

平台价值在"省掉自建 NLP 流水线"：直接提供词频、句频、相似词（基于词向量,常用于扩展自定义词典）、语调、风险、可读性、文本相似性等预计算指标，输出 Excel/CSV/纯文本，可直接接 Stata。国内文本分析论文里"年报词频法"构造的指标（数字化转型、管理者短视、绿色转型词频等），相当一部分走的就是这类平台。

## Fields

[[胡楠-2021-管理者短视主义-长期投资-文本分析]] 实际使用的处理链：

- PDF → TXT 转换；表格剔除（表格模板化严重、增量文本信息少）。
- 扫描件与缺失文件剔除。
- MD&A 章节抽取（正则定位"管理层讨论与分析"章标题）。
- 中文分词：Jieba 通用词典 + 《英汉现代财会大词典》等财经专用词典。
- 去停用词；词向量存储（Word2Vec 下游建模）。

## Merge Keys

`stkcd` + `year`（报告年度）。

## Cleaning Rules

- 同 CSMAR 惯例：剔除金融保险、ST/*ST、上市不足两年样本。
- 文本侧额外剔除扫描版年报与 MD&A 缺失样本。
- 字符编码统一 UTF-8，繁简转换提前完成。
- 用词频法造指标时，**词典构造过程要在论文中完整披露**（种子词 → 相似词扩展 → 人工筛除），这是审稿人最常攻击的环节。

## Missingness

- 2007 年前部分公司未单独披露 MD&A 章节。
- 早期年份扫描件比例高，样本流失明显。

## Access

- 官网 `www.wingodata.cn`，签约高校校园 IP 免密登录；多所财经院校已订购或开通试用。

## Project Files

WinGo 是商业平台，本项目未直接落盘其输出。胡楠等 (2021) 公开的 Myopia_Index 已存于 `03_原始数据/管理者短视主义_2007-2024`，可绕过自建流程直接使用。
